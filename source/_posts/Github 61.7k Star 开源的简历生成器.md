    大家好，我是牛皮糖！今天在GitHub看到一款简历生成工具，给定对应的内容就能生成好看的简历，而且还有在线网站可以体验。GitHub上拥有61787星的开源项目——Resume。

**
**

**项目介绍**

Resume是一个开源的简历生成器，它提供了一个简单易用的界面，让你可以快速创建一份专业的简历。无论是求职、升学还是其他任何需要展示个人能力的场景，Resume都能助你一臂之力。

![](https://mmbiz.qpic.cn/mmbiz_gif/p1ESIQQvfrTsRCQAK7JEiapq8zce18gGLtEAfRVXRicBBcBFLc2VfwiaUpjenRjWticed5j9ecoxaAcJZOrBWrh5Gw/640?wx_fmt=gif&from=appmsg)

**
**

**核心功能**

- 模板丰富：Resume提供了多种简历模板，从简洁明了到专业详尽，总有一款适合你。

- 在线编辑：用户可以直接在网页上编辑简历内容，无需下载任何软件。

- 实时预览：编辑过程中，Resume会实时展示简历的最终效果，方便你随时调整。

- 导出功能：完成编辑后，可以导出PDF、Word等多种格式的简历文件。

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrTsRCQAK7JEiapq8zce18gGLn6sFt3QZrC87Ur3h58Q4qMibfVo5xDckFibeo31pib5uicXjqjuRklYE6g/640?wx_fmt=png&from=appmsg)

****

访问Resume的GitHub页面：Resume

点击"Try it now"按钮，开始在线编辑简历。

选择一个你喜欢的模板，然后开始填写个人信息、教育背景、工作经历等内容。

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrTsRCQAK7JEiapq8zce18gGLEYNHAibOFiaecasN0pkXvHKlJsibB6kib3ovhNAB7a45h2z3uibPib4VFyYw/640?wx_fmt=png&from=appmsg)

编辑完成后，点击"Preview"查看简历效果。

如果满意，点击"Download"下载你的简历。

****

如果你对Resume感兴趣，或者想要体验一下这个强大的简历生成器。

如果不想在线使用也可以将项目克隆下来然后进行 docker 安装。

- 
`git clone https://github.com/xitanggg/open-resume.git`

- 更改目录cd open-resume

- 构建容器docker build -t open-resume .

- 启动容器docker run -p 3000:3000 open-resume

- 打开浏览器并访问http://localhost:3000即可实时查看 OpenResume
