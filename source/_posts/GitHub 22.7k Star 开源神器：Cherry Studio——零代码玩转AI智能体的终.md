### **📌 5分钟极速上手**
**步骤1：安装Cherry Studio**`# 官网下载对应版本（Win/macOS/Linux）
https://cherry-ai.com/`**步骤2：配置MCP服务器（以arXiv论文助手为例）**- 1. 安装依赖工具：```
`brew install uv  # macOS
uv tool install arxiv-mcp-server  # 安装arXiv服务`
```

- 2. 在Cherry Studio中添加MCP服务器：

- • 类型：`Stdio`

- • 命令：`uv tool run arxiv-mcp-server`

- • 存储路径：`~/Documents/arxiv`

**步骤3：使用AI自动化**输入指令：

"搜索'大语言模型优化'的最新5篇论文，下载PDF并生成中文摘要"

AI将自动完成：arXiv搜索→下载→调用DeepSeek解读→保存Markdown
