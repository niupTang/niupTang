### Real-Time-Voice-Cloning

- GitHub 链接：https://github.com/CorentinJ/Real-Time-Voice-Cloning

- 主要功能：这个项目能够克隆一个人的声音并在实时中生成任意语音。它通过录制一个人的声音样本，然后使用这些样本来生成新的语音。

- 技术特点：

- 克隆声音只需5秒钟的录音。

- 实时生成语音。

- 使用深度学习技术来建模声音特征。

### MockingBird

- GitHub 链接：

https://github.com/babysor/MockingBird

- 主要功能：MockingBird 是一个 AI 拟声工具，能够在5秒内克隆声音并生成任意语音内容。它支持中文普通话，并且在多个数据集上进行了测试。

- 技术特点：

- 支持中文，包括多个中文数据集。

- 适用于 PyTorch，测试版本为 1.9.0。

- 可在 Windows 和 Linux 操作系统上运行，包括 M1 芯片的 macOS。

- 提供了易于使用的合成器效果，通过重用预训练的编码器/声码器。

- 支持 Web 服务器，可以远程调用生成结果。

**
**

    Real-Time-Voice-Cloning 主要针对英文，而 MockingBird 特别支持中文普通话。

    总的来说，两个项目都提供了声音克隆的能力，但 MockingBird 在中文支持、社区贡献和 Web 服务方面提供了更多的特性和便利。根据你的具体需求和偏好，你可以选择适合你的项目。
