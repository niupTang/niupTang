* 戳上方蓝字“**牛皮糖不吹牛**”关注我

    大家好，我是牛皮糖！最近在研究如何高效整合多语言模型（LLM）资源时，我发现了一个非常实用的开源项目——**Cherry Studio**。对于像我这样需要频繁使用不同语言模型进行创作和研究的人来说，Cherry Studio 简直就是一款“神器”。

### 一、多语言模型支持，一站式搞定！
**Cherry Studio** 是一款桌面客户端，支持 **Windows**、**Mac** 和 **Linux** 系统。它最大的亮点就是可以接入多种语言模型服务，包括但不限于 **OpenAI**、**Gemini**、**Anthropic** 等主流云服务，还能接入 **Claude**、**Peplexity**、**Poe** 等 AI 网络服务，甚至支持本地模型（如 **Ollama**）。这意味着你可以在一个客户端里，轻松切换不同的语言模型，根据需求选择最适合的工具。

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrQTKgYnAMSKMaVIficJSibcXicibr2fiaqbViaYUlria7jlsxtOOD9qGIZuR7etuuPnGMkkC2DCxDzO25Ldg/640?wx_fmt=png&from=appmsg)

### 二、功能强大，满足多样化需求！
Cherry Studio 的功能非常丰富，让我来一一介绍：

1. **AI 助手与对话**- • 内置了 **300+ 预配置的 AI 助手**，你可以直接使用这些助手进行对话，获取灵感或者解决问题。

- • 支持自定义创建 AI 助手，打造专属的智能助手。

- • 支持多模型同时对话，你可以同时和多个语言模型进行交流，对比它们的回答，找到最适合你的解决方案。

2. **文档处理**- • 无论是文本、图片、Office 文档还是 PDF 文件，Cherry Studio 都能轻松处理。

- • 支持 **WebDAV 文件管理与备份**，让你的文件管理更加便捷。

- • 通过 **Mermaid 图表** 进行数据可视化，代码语法高亮功能也让代码阅读更加清晰。

- ![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrQTKgYnAMSKMaVIficJSibcXicQPD0VgkWjzP6BPSApFZa3ib0mWFRTziadNZyPXmJg5YIKugyCrh6hT2A/640?wx_fmt=png&from=appmsg)

3. **实用工具集成**- • **全局搜索** 功能让你可以快速找到所需内容。

- • **话题管理系统** 帮助你更好地组织知识。

- • **AI 助力的翻译功能** 让你能够轻松跨越语言障碍。

- • **拖拽排序** 功能让界面更加个性化。

- • 支持 **小程序**，进一步拓展了功能边界。

### 三、使用教程
Cherry Studio 的使用非常简单，无需复杂的环境配置。你可以直接下载安装包，安装完成后即可使用。在使用过程中，你可以通过简单的界面操作，快速接入不同的语言模型，并开始对话或文档处理。

1. **安装 Cherry Studio**- • 访问 GitHub - CherryHQ/cherry-studio 下载安装包。

- • 按照提示完成安装。

2. **配置 DeepSeek 的 API 密钥**- • 复制 DeepSeek 的 API 密钥。

- • 将密钥粘贴到 Cherry Studio 的 API 密钥配置页面。

- • 点击“校验”按钮，确保配置成功。

- ![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrQTKgYnAMSKMaVIficJSibcXicNpykPKqibxzDTBmIzqmAG3W4RMbBEopQsgdvfGk6iarR5CknSa1t5fSg/640?wx_fmt=png&from=appmsg)

3. **开始使用**- • 选择你需要的语言模型，开始对话或文档处理。

- • 利用全局搜索和话题管理系统，快速找到所需内容。

### 五、总结工具优势
Cherry Studio 给我带来了极大的便利。它不仅整合了多种语言模型，还提供了强大的文档处理和实用工具功能。无论是创作、研究还是日常学习，它都能满足我的多样化需求。如果你也在寻找一款高效的语言模型工具，Cherry Studio 绝对值得一试！

你可以通过访问 GitHub - CherryHQ/cherry-studio 获取更多详情和安装包。感谢大家的支持，希望 Cherry Studio 能成为你创作和研究的好帮手！
