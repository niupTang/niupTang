大家好，我是牛皮糖！今天给大家介绍一款在 GitHub 上斩获 **6.8K+ Star** 的神器——**doocs/md 微信 Markdown 编辑器**。如果你还在为公众号排版头疼，这款工具绝对能让你效率翻倍！

## 🎯 项目介绍

**doocs/md** 是一款高度简洁的微信 Markdown 编辑器，支持 Markdown 语法、自定义主题样式、内容管理、多图床、**AI 助手**等特性。

**一句话总结：Markdown 文档自动即时渲染为微信图文，让你不再为微信内容排版而发愁！**

> 🔗 在线体验：https://md.doocs.org

推荐使用 Chrome 浏览器，效果最佳。

## ✨ 核心功能特性

### 🎨 编辑能力

| 功能 | 说明 |
|------|------|
| ✅ 完整 Markdown 支持 | 支持所有基础语法、数学公式 |
| ✅ 图表渲染 | 支持 Mermaid 图表和 GFM 警告块 |
| ✅ PlantUML 支持 | 强大的 UML 图表渲染 |
| ✅ Ruby 注音扩展 | 支持 `[文字]{注音}` 格式 |
| ✅ 代码高亮 | 丰富的代码块高亮主题 |

## 🛠️ 快速部署（三种方式）

### 方式一：直接在线使用

访问 https://md.doocs.org 即可，无需任何配置！

### 方式二：npm cli 私有部署

```bash
# 安装
npm i -g @doocs/md-cli

# 启动
md-cli

# 访问 http://127.0.0.1:8800
```

### 方式三：Docker 一键部署

```bash
docker run -d -p 8080:80 doocs/md:latest
```

然后访问 http://localhost:8080 即可！
