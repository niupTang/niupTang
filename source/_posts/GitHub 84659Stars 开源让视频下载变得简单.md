**yt-dlp** 是一个命令行程序，它可以帮助我们从YouTube等视频平台下载视频。它是基?*youtube-dl**的分支，继承了youtube-dl的所有特性，并在此基础上进行了扩展和优化?*yt-dlp**支持的网站非常广泛，几乎涵盖了所有主流的视频分享平台?

**如何使用?* 

使用**yt-dlp**非常简单。首先，你需要在你的计算机上安装Python环境。然后，你可以通过pip安装**yt-dlp**?

`pip install yt-dlp`安装完成后，你就可以使用以下命令来下载视频了?

`yt-dlp [URL]`其中`[URL]`是你想要下载的视频的链接?

当然还有查看帮助命令

`yt-dlp [OPTIONS] [--] URL [URL...]
