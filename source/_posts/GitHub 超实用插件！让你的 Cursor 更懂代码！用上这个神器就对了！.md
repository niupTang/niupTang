## ✨ 项目简介：什么是 `repomix`？
`repomix` 是一个代码目录解析的增强工具，它的核心能力是：

📦 **将你整个代码仓库的上下文加载到 AI 中**，让 Cursor 真正读懂你的项目结构、依赖关系、文件逻辑。

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrRSPFoqg4eBh56XfNFX734Lshn88b9iav9RVzWa6elAVR1HP3PHeaEMd33cfLtxX66odiaNMBXcTaww/640?wx_fmt=png&from=appmsg)

换句话说，它能解决一个根本问题：

✨ **AI 虽然强，但不了解全局结构，回答经常答非所问？用上 repomix，它就能看懂整个项目脉络！**

## 🧠 repomix 能帮你做什么？
• 📝 **精准问答**：让你对项目中任意模块发问，AI 立即理解上下文给出靠谱回复。
• 🔍 **智能搜索**：快速定位项目文件、函数、变量定义，代码迷路？不存在的。
• 💡 **上下文增强**：大大提升 Chat 的“代码感知能力”，不再只看你当前文件。
• 🛠️ **开发协作**：团队成员接手项目也能快速掌握全局结构，一起用更高效。

## 📦 如何集成到 Cursor 中？
- 1. 打开你的项目目录，确保项目结构合理。

- 2. 安装 repomix 插件：`bun install -g repomix`如果你还没装 `bun`，记得先装一下。

- 3. 在项目根目录运行：```
`repomix --style  markdown`
```
它会分析你的项目结构并生成一份上下文索引。

- 4. 复制生成的索引路径，在 Cursor 中设置你的Rules ，要他根据你的文件结构来进行分析你的项目。
