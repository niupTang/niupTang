# **PySpur 是什么？**
PySpur 是一个基于 **Python** 的开源可视化AI智能体开发平台，专为简化AI工作流构建而生。它通过**图形化拖拽**替代传统编码，让非技术用户也能快速搭建复杂的AI应用，同时支持开发者深度定制模块。

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrTecSDvzADK0FicVt7n18uTGibFjXsFEx0OExly3rtOf01Buc5SzKax1ZhNVZcH8HantA0pytMdhaCQ/640?wx_fmt=png&from=appmsg)

它支持构建工作流、运行测试用例、记忆管理、文件上传、结构化输出、RAG、多模态和评估等功能，同时兼容 100 多家大模型服务供应商。

### **核心亮点**
✨ **拖拽式开发**：无需写代码，通过节点连线即可构建AI工作流（如对话系统、文档分析等）。
✨ **多模态支持**：兼容文本、图像、音频、视频处理，适合复杂数据分析任务。
✨ **RAG 增强**：内置文档分块、向量嵌入功能，轻松对接检索增强生成模型。
✨ **工具集成**：直接调用Slack、Google Sheets等第三方服务，扩展工作流能力。
✨ **实时调试**：每一步执行结果可视化，支持即时调整和优化。

# **快速上手 PySpur**
### **安装方法**
- 1. **Python 环境**（需3.8+）：`pip install pyspur`

- 2. **启动可视化界面**：```
`pyspur-ui`
```
首次运行会自动打开浏览器访问 `http://localhost:8501` 。

### **基础教程**
- 1. **新建工作流**：拖入“输入节点”接收用户问题，连接“LLM调用节点”生成回复。

- 2. **调试技巧**：右键节点→“实时预览”，查看中间结果。

- 3. **部署为API**：完成开发后，一键导出为FastAPI服务。

# **注意事项**
⚠️ **模型依赖**：需自行配置OpenAI/Claude等API密钥（支持100+供应商）。
⚠️ **性能优化**：复杂工作流建议启用“异步执行”模式，避免阻塞。
⚠️ **中文支持**：界面暂以英文为主，但可处理中文数据。
