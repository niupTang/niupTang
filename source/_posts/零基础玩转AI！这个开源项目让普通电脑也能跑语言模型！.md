*** 戳上方蓝字“牛皮糖不吹牛**”关注我

**家人们，我是牛皮糖！** 最近发现个让编程小白都能装X的神器——GitHub上6.2k星爆火的 **MiniMind**。简单说就是：**用打游戏的电脑，花3小时就能训练出会聊天的AI**！

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrSzZyGYIhmibK7hSlXGgQvAvogC2OZqtH7V65AB9oE8MORicr78vCibqvjgeo0O9OkcsvbJmOCGXHtPA/640?wx_fmt=png&from=appmsg)

### 🔥 软件 
- **下软件像装APP**
直接复制两行代码，自动下载所有需要的程序包，连数据集都帮你整理好了（中文语料自带火星文过滤器）

- 2. **训练像煮泡面**
运行命令后去吃个饭，回来就看到AI开始憋诗了：`# 做饭前输入（支持中文！）
python train.py --model_name "我的第一个AI"
# 吃完饭后看到
[输出] AI生成："床前明月光，火锅要鸳鸯"（虽然不押韵但能吃）`

- 3. **改参数像调美颜**
显卡差的把`batch_size`调小（相当于手机开省电模式）
想玩大的加`--max_length 512`（让AI小作文能写500字）

### 💡 三步极简教程（小学生版）
```
`# 1. 复制这个（就像下载微信）
git clone https://github.com/jingyaogong/minimind.git

# 2. 装依赖（自动安装所有需要的软件）
pip install -r requirements.txt

# 3. 开训！（建议取个中二名字）
python train.py --model_name "牛皮糖のAI分身"`
```
### ⚠️ 新手必看避坑指南
- • **显卡选择**：
3060就能跑（和吃鸡配置差不多）
没有显卡？用CPU也能玩（就是慢得像树懒）

- • **中文优化**：
遇到乱码就去文档第15页，作者给了火星文过滤模板

- • **效果预期**：
别指望取代ChatGPT！这就是个"AI婴儿"——
✅ 能背唐诗 ✅ 会编冷笑话 ❌ 解不开高数题
