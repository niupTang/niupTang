# Qlib 是什么
Qlib 是微软开源的一站式量化投资平台，**从数据处理、特征工程、模型训练到回测、策略部署全链路覆盖**，简直就是“量化投资的全能工具箱”。不需要你有深厚的金融背景，也不需要昂贵的付费服务，**你只需要一台电脑，就能从零打造属于你的智能交易策略**。

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrTH54icichROCgkRG2OZ6pnb8FW3ykwblyx6IAcic2JgeANZ2cgPTyyM77M50ChI5z3KvqbeBTtuRakA/640?wx_fmt=png&from=appmsg)

# **核心功能亮点**
**1. 数据处理 + 特征工程一步到位** Qlib 内置了多种金融数据源，支持分钟级别数据处理，并提供自动化特征提取工具，降低数据预处理门槛。

**2. AI 驱动的策略模型训练** 不仅支持传统的因子模型，还集成了 LSTM、Transformer、GAT 等深度学习模型，助你快速打造智能选股策略。

**3. 灵活强大的回测系统** 内置回测引擎可轻松模拟历史交易情况，支持自定义滑点、手续费等参数，帮助你检验策略有效性。

**4. 可扩展性极强** 你可以根据自己的需求，灵活接入自定义数据源、自定义因子和模型，Qlib 提供了完整的 API 接口支持。

过去量化投资被大机构垄断，普通人想要入门，动辄就是几十万的系统费用和专业门槛。 而 **微软 Qlib 开源之后，彻底打破了这层“技术壁垒”**，让普通投资者也能低成本地学习和应用量化策略，**真正实现用 AI 为自己的财富加速**。

# **如何快速上手 Qlib？**
- 1. 访问项目地址：https://github.com/microsoft/qlib

- 2. 克隆代码仓库，并根据文档下载示例数据集。

- 3. 通过 Python 环境运行官方 Demo 策略，体验完整的量化策略训练与回测流程。

- 4. 结合自己的数据与想法，自定义打造属于你的投资策略。

**一、环境准备**

- 1. **建议使用 Conda 虚拟环境：**`conda create -n qlib python=3.10 -y
conda activate qlib`

- 2. **升级 pip 和 Cython：**```
`pip install --upgrade pip cython`
```

### 方式一：直接安装稳定版
```
`pip install pyqlib`
```
### 方式二：安装最新开发版（推荐）
```
`git clone https://github.com/microsoft/qlib.git
cd qlib
pip install -e .[dev]`
```
**三、下载示例数据**

`wget https://github.com/chenditc/investment_data/releases/latest/download/qlib_bin.tar.gz
mkdir -p ~/.qlib/qlib_data/cn_data
tar -zxvf qlib_bin.tar.gz -C ~/.qlib/qlib_data/cn_data --strip-components=1
rm -f qlib_bin.tar.gz`**四、运行示例（LightGBM回测）**

- 1. 切换到示例目录：`cd examples`

- 2. 运行 benchmark 流程：```
`qrun benchmarks/LightGBM/workflow_config_lightgbm_Alpha158.yaml`
```

- 3. 成功后，你将看到类似：```
`'The following are analysis results of the excess return without cost.'
                        risk
mean               0.000708
std                0.005626
annualized_return  0.178316
information_ratio  1.996555
max_drawdown      -0.081806`
```

**五、可选：可视化分析报告**

- 1. 安装分析依赖：`pip install .[analysis]`

- 2. 打开示例 Jupyter Notebook：```
`jupyter notebook examples/workflow_by_code.ipynb`
```

**常见问题**

- 1. **Python版本**：支持 Python 3.8 - 3.12（推荐 3.10）

- 2. **M1芯片 Mac 用户**：安装 LightGBM 可能需要：`brew install libomp`
