# 开源版“微信Coze”来了？ChatWiki：解锁公众号的超级 Agent 形态

大家好，我是太阳鸟。

运营公众号的朋友可能有同感：微信原生后台的自动回复功能实在**太弱了**。

想搞个 AI 客服？想连接某个 API？想根据关键词触发不同的业务流？
如果不购买昂贵的第三方工具，或者不想把数据交给第三方，唯一的出路似乎只有**自己开发**。但认证服务号的要求又把 90% 的个人号挡在门门外。

今天要推荐的这个开源项目 **ChatWiki**，堪称**垂直领域的 Coze + n8n**，它能零成本把你的公众号变成一个超级 AI 智能体。

![ChatWiki 概念图：连接微信与超级大脑](/images/chatwiki/concept_brain.png)

---

### 1. 为什么它是“效率核弹”？

**ChatWiki** 不仅仅是一个 AI 客服，它是一个基于 RAG（检索增强生成）和工作流（Workflow）的**全能 Agent 平台**。

最让我震惊的是它的一个核心突破：

> **全行业首创：支持未认证的订阅号（个人号）！**

这意味着，哪怕你只是一个普通的个人博主，也能拥有以前只有企业服务号才能具备的 AI 能力：
- **私信自动回复**（文本、图片、语音、视频全支持）。
- **智能标签管理**（根据用户对话自动打标）。
- **工作流编排**（完全可视化的拖拽式操作）。

### 2. 核心功能拆解

#### 🛠️ 1. 微信生态的深度集成
ChatWiki 像是为微信量身定做的。它不仅支持公众号，还能通吃**微信客服、微信小店客服**。
通过 hack 掉了微信的限制，它让个人号也能实现复杂的业务逻辑。比如用户发送“合作”，自动回复商务联系方式并通知管理员；用户发送一张图，AI 自动进行 OCR 识别并返回结果。

#### 📊 2. RAG 知识库：让 AI 懂你的历史
不需要费力整理 QA 文档。ChatWiki 支持一键抓取你公众号的历史文章，自动建立知识库。
无论是粉丝问你的过往观点，还是咨询产品细节，AI 都能基于你的内容库，给出准确、有温度的回答。

#### 🤖 3. 可视化工作流 (Workflow)
这是它最像 Coze/n8n 的地方。
你可以像搭积木一样编排业务流程：
- **触发器**：收到关键词“日报”。
- **动作1**：调用 API 获取今日科技新闻。
- **动作2**：让 DeepSeek 模型总结成 300 字简报。
- **动作3**：推送给用户。

全过程不需要写一行代码。

![ChatWiki 可视化工作流界面](/images/chatwiki/workflow_ui.png)

### 3. 一键部署，数据私有

相比于把粉丝数据交给第三方平台，ChatWiki 最大的优势是**私有化**。基于 Docker，你可以在自己的服务器上极速部署。

```bash
# 1. 下载项目
git clone https://github.com/zhimaAi/chatwiki.git

# 2. 启动服务 (内置 Docker Compose)
cd chatwiki/docker
docker compose up -d

# 3. 访问后台
http://localhost:18080
```

默认账号：`admin` / `chatwiki.com@123`

### 4. 适合谁？

- **个人开发者**：想给自己的公众号加点 AI 魔法，但不想折腾微信认证。

- **技术极客**：喜欢捣腾 n8n、Dify 等工具，想在微信生态里复刻一套 Agent 流程。
