* 戳上方蓝字“**牛皮糖不吹牛**”关注我

    大家好，我是牛皮糖！今天要介绍的是一个在GitHub上拥有25k星的开源项目——LosslessCut。这是一个非常实用的视频编辑工具，它能够让你无损地剪切视频，就像剪切图片一样简单。

![](https://mmbiz.qpic.cn/mmbiz_jpg/p1ESIQQvfrS4vkeTQJ9F6axib3SFOzxQBfeicyXPibX3zPbXdqBaXsic55GSzw86iaQJls1PYLYH0ia6yftL88FHEfJA/640?wx_fmt=jpeg&from=appmsg)

**1****、特征**

- 大多数视频和音频格式的无损剪切

- 智能剪切（实验性）

- 无损剪切视频/音频的部分内容（用于剪切广告等）

- 无损重新排列视频/音频片段的顺序

- 无损合并/连接任意文件（具有相同的编解码器参数，例如来自同一台摄像机）

- 无损流编辑

- 组合来自多个文件的任意音轨（例如，向视频文件添加音乐或字幕音轨）

- 删除不需要的曲目

- 仅替换或重新编码某些曲目

- 从文件中提取所有轨道（将一个文件的视频、音频、字幕、附件和其他轨道提取到单独的文件中）

- 快速的多文件工作流程（注意：尚不支持批量导出）

- 键盘快捷键工作流程

- 无损地将视频/音频重新混合到不同的容器（文件）格式

- 从 JPEG/PNG 格式的视频中截取全分辨率快照（低质量或高质量）

- 将视频帧范围导出为图像（每 n 帧、每秒、按场景变化、最佳缩略图）

- 仅从选定的时间范围（片段）导出

- 可选择在图像文件名中包含原始时间戳

- 手动输入截点时间

- 应用每个文件的时间码偏移（并从文件自动加载时间码）

- 编辑文件元数据、每个轨道的元数据和每个轨道的配置

- 更改视频中的旋转/方向元数据

- 查看所有曲目的技术数据

- 时间轴缩放和帧/关键帧跳跃，用于剪切关键帧

- 视频缩略图和音频波形

- 将每个项目的剪切片段保存到项目文件中

- 查看 FFmpeg 最后的命令日志，以便您可以在命令行上修改并重新运行最近的命令

- 撤消/重做

- 为剪切片段添加标签

- 使用标签注释片段

- 导入/导出片段：MP4/MKV 章节标记、文本文件、YouTube、CSV、CUE、XML（DaVinci、Final Cut Pro）等

- MKV/MP4 嵌入式章节标记编辑器

- 查看字幕

- 可自定义的键盘热键

- 黑场检测、静音音频检测、场景变化检测

- 将时间线分成长度为 L 的段或 N 段，甚至是随机段！

- 加速/减慢视频或音频文件的速度（改变 FPS）

- 基本CLI和HTTP API

- 在地图上显示（DJI）嵌入式 GPS 轨迹

- 通过 HTTP（例如 HLS .m3u8）无损下载视频
