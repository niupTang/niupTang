* 戳上方蓝字“**牛皮糖不吹牛**”关注我

    大家好，我是牛皮糖！今天，我要给大家介绍一个在GitHub上拥有26000 Star的开源项目——**DeepFaceLive**。

**    DeepFaceLive**是一个基于深度学习的实时人脸交换工具，它能够将视频中的人物面部实时替换成另一个人的脸。这项技术在电影制作、游戏开发以及虚拟现实等领域有着广泛的应用前景。

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrTOmymGsg6IibeeXz4TybASfNROBlWqC3CNMldH25oNByudMahHrKmoR0QRbX48PYBNqziaPic0icluAw/640?wx_fmt=png&from=appmsg)

    DeepFaceLive 应用中还有一个 Face Animator 模块。您可以使用视频或摄像头中的自己的脸部来控制静态脸部图片。质量不是最好的，并且需要对每对脸部进行精细的脸部匹配和调整参数，但对于搞笑视频和表情包或使用 35 TFLOPS GPU 以 25 fps 的速度进行实时流式传输来说已经足够了。

**安装环境******

任何兼容 DirectX12 的显卡

(推荐 RTX 2070+ / Radeon RX 5700 XT+ )

具有 AVX 指令的现代 CPU

4GB RAM，32GB+分页文件

Windows 10
