**Shardeum是什么？**

Shardeum是一个创新的EVM兼容区块链平台，利用动态状态分片实现前所未有的可扩展性。通过实施分片模型，Shardeum确保更快的处理时间和更低的交易成本，同时又不损害安全性或去中心化。

**核心特征**

- • **可扩展性**：通过分片实现水平可扩展性。

- • **高性能**：低延迟、高吞吐量。

- • **安全性**：先进的加密技术和强大的共识协议。

- • **去中心化**：真正去中心化的网络，没有单点故障。

- • **互操作性**：现有DApp和智能合约的EVM兼容性。

**如何使用Shardeum**

对于程序员来说，要如何安装或使用Shardeum项目呢？以下是一些基本步骤：

- 1. **安装先决条件**：确保你的开发环境中安装了Node.js (v18.19.1)、npm (v10.2.4)、Rust (v1.74.1)。

- 2. **设置环境**：按照官方文档设置你的本地环境。

- 3. **安装Shardeum**：

`git clone https://github.com/shardeum/shardeum.git
cd shardeum
npm ci`
- 4. **网络配置**：

`git apply debug-10-nodes.patch`
- 5. **编译项目**：

`npm run prepare`
- 6. **安装Shardeus CLI**：

`npm install -g shardus
npm update @shardus/archiver`
- 7. **本地运行网络**：

`shardus start 10`
- 8. **运行JSON-RPC服务器**：

`git clone https://github.com/shardeum/json-rpc-server.git
cd json-rpc-server
npm install
npm run start`
- 9. **使用MetaMask进行测试**：将Shardeum网络添加到MetaMask，并配置Genesis文件以获取测试代币。
