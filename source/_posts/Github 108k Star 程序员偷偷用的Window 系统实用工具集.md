### **1. 始终置顶（Always on Top）**
允许用户通过快捷键将任何窗口固定在屏幕最前端，方便多任务处理。

### **2. 颜色选择器（Color Picker）**
一个简单的系统范围的颜色选择器，可以通过快捷键激活，方便用户从屏幕上的任何位置选择颜色并获取其十六进制或 RGB 值。

### **3. FancyZones**
一个窗口管理器，允许用户创建复杂的窗口布局，并将应用程序窗口快速对齐到这些布局中，提高多窗口工作效率。

### **4. 文件资源管理器加载项（File Explorer Add-ons）**
为文件资源管理器添加预览窗格功能，支持预览 SVG 图标文件和 Markdown 文件。

### 5. 图像大小调整器（Image Resizer）
一个右键菜单扩展，允许用户快速调整图片大小，支持批量处理。

### **6. 键盘管理器（Keyboard Manager）**
允许用户重新映射键盘按键和快捷键，以适应个人使用习惯。

### **7. 鼠标实用工具（Mouse Utilities）**
包括鼠标荧光笔和查找我的鼠标功能，帮助用户在屏幕上更容易地找到鼠标指针。

### **8. 批量重命名（PowerRename）**
一个强大的批量重命名工具，支持正则表达式，可以通过右键菜单快速访问。

### **9. 快捷键指南（Shortcut Guide）**
通过按下 Windows 键一段时间，显示一个覆盖层，列出当前活动应用程序的常用 Windows 快捷键。

### **10. 文本提取器（Text Extractor）**
一个 OCR 工具，允许用户从屏幕上的任何图像中提取文本，支持多语言。

### **11. 视频会议静音（Video Conference Mute）**
允许用户通过快捷键快速静音和取消静音摄像头和麦克风，适用于视频会议。

### **12. PowerToys Run 12. PowerToys 运行**
一个快速的启动器，可以通过快捷键激活，支持搜索应用程序、文件、计算器、浏览器书签等，类似于 macOS 的 Spotlight。

### **13. 窗口增强（Window Enhancements）**
包括窗口捕捉和窗口切换器，帮助用户更高效地管理窗口。

### **14. 黑暗模式切换（Dark Mode Toggle）**
允许用户快速切换系统主题为黑暗模式。
