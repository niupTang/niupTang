* 戳上方蓝字“**牛皮糖不吹牛**”关注我

    大家好，我是牛皮糖！今天要给大家带来一个超级实用的开源项目——GoSpeed。这个项目是由一群热爱技术的开发者们精心打造的，它是一个高速下载器，支持HTTP、BitTorrent、Magnet等多种协议，几乎涵盖了我们日常下载的所有需求。而且，它还支持全平台，无论是Windows、macOS、Linux，还是Android和iOS，都能轻松使用。

![](https://mmbiz.qpic.cn/mmbiz_jpg/p1ESIQQvfrSlJWp6Zj127uS5GrPu9QaNKZk9hthOw36fRd6XOgiaX9g6VxatLNfLswSBv5aibiaYywagAzh3APd4A/640?wx_fmt=jpeg&from=appmsg)

    GoSpeed的厉害之处在于它的高度可定制性。它不仅提供了基本的下载功能，还允许用户通过API集成或Flutter开发的扩展来实现更多个性化的功能。这意味着，无论你是想要一个简单的下载工具，还是想要一个功能丰富的下载管理器，GoSpeed都能满足你的需求。

多平台支持：

-  Windows

-  Macos

-  Linux

-  Android

-  iOS

-  Web

-  Docker

-  Command Tool

支持 HTTP 和 HTTPS

支持 BitTorrent 和 Magnet

种子、DHT、PEX、uTP、Webtorrent、Upnp

每天自动更新跟踪链接列表

国际化

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrSlJWp6Zj127uS5GrPu9QaNiauoFibBmicqZ0NLGSxDVpXssueeibWk7ZgRDBxSNrJJbUkbiaRLKqdRaVQ/640?wx_fmt=png&from=appmsg)

对于程序员来说，GoSpeed的安装和使用都非常简单。你可以通过go install命令直接安装GoSpeed的命令行工具，也可以通过浏览器扩展来增强你的下载体验。此外，如果你想要为GoSpeed贡献代码，项目的前端代码位于ui/flutter目录中，后端则使用Golang开发，两者通过HTTP协议进行通信。
