* 戳上方蓝字“**牛皮糖不吹牛**”关注我

    大家好，我是牛皮糖！程序员也顾家 ，Grocy 是一个开源的、自托管的全功能 ERP 系统，专注于管理家庭库存、购物清单和食谱。它旨在帮助用户更有效地管理他们的食品库存，避免浪费，并简化购物过程。以下是关于 Grocy 项目的详细介绍：

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrSqug9UXHklJPllPtKEVfmicBszjlRE7sUJH96icpYQjnCQ59w9ibZoKoB0dyk6SZjvGOf7uBaKrwSZQ/640?wx_fmt=png&from=appmsg)

项目特点

- **库存管理**：跟踪食品和其他物品的库存水平，自动计算过期日期。

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrSqug9UXHklJPllPtKEVfmicB9DU2GxRxS0jPNEh2Uk9L97dFGGT9zltAT7H3vUcoFEDTtbTib978ow/640?wx_fmt=png&from=appmsg)

- **购物清单**：根据库存水平自动生成购物清单。

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrSqug9UXHklJPllPtKEVfmicZ0LQsMN6Cx8XQkccCK1W6R4HZBFg3xajsyV9fklX2YDYhwfBgrNmXA/640?wx_fmt=png&from=appmsg)

- **食谱管理**：保存和组织食谱，包括所需的食材和步骤。

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrSqug9UXHklJPllPtKEVfmicFc5eLcicjW6EJ50e4GLBBrNJJKvHM1awAviaExT8E8vIbfvT87PeadicA/640?wx_fmt=png&from=appmsg)

- **批量转换**：支持不同单位之间的转换，例如克、升等。

- **任务管理**：设置周期性任务，如清洁或检查库存。

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrSqug9UXHklJPllPtKEVfmicHwR0JcVksKzfVibEn2d3JMwCKyXNDVFb3Lia76lNYL8iaaSvVU63PvmWg/640?wx_fmt=png&from=appmsg)

- **多用户支持**：允许多个用户访问和更新库存信息。

- **API 支持**：提供 REST API，方便与其他系统集成或自动化任务。

技术栈

- **前端**：使用 Vue.js 和 BootstrapVue 构建响应式用户界面。

- **后端**：使用 PHP 编写，提供 REST API 和业务逻辑。

- **数据库**：支持多种数据库，包括 MySQL、PostgreSQL 和 SQLite。

- **身份验证**：支持基本的 HTTP 认证和 JWT（JSON Web Tokens）。

安装和部署

Grocy 可以部署在各种环境中，包括 Docker、Linux 服务器或 Windows 系统。项目提供了详细的安装指南，帮助用户根据自己的需求进行设置。

社区和贡献

Grocy 拥有一个活跃的社区，用户可以通过 GitHub 提交问题、功能请求或贡献代码。项目维护者积极地与社区互动，不断改进和更新软件。

使用场景

Grocy 适用于需要管理家庭库存和购物清单的用户，特别是那些希望减少食品浪费、提高生活效率的人。它也适用于那些喜欢烹饪和尝试新食谱的人。

开源许可

Grocy 是在 MIT 许可证下发布的，这意味着它可以被自由地使用、修改和分发。

项目地址：

**https://github.com/grocy/grocy 来获取更多信息，包括源代码、文档和安装指南。**
