### ✅ 第一步：安装 OpenBB 平台
确认你的 Python 环境是 `3.9.21 ~ 3.12`（推荐使用虚拟环境）。

然后一条命令安装完整版平台功能：

`pip install "openbb[all]"`这一步会安装所有必备模块，包括你熟悉的股票、ETF、宏观数据、量化工具等等。

### ✅ 第二步：运行 API 服务器
接下来，一条命令开启 OpenBB 的后端服务器（基于 FastAPI + Uvicorn）：

`openbb-api`启动后默认运行在：

`http://127.0.0.1:6900`你可以直接在浏览器里打开这个地址，看到 OpenBB 的 API 文档页面，证明服务器已经成功启动！

### ✅ 第三步：连接到 OpenBB Workspace
打开浏览器，登录你本地或在线的 OpenBB Workspace 页面。

然后依次操作：

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrRclFMfzh1qvibsalNxO4EmicSJasw0qnFEkxjOwb20rPYtlNpR8J1FXbTgY4Nb334tbntLgZtxhr6w/640?wx_fmt=png&from=appmsg)

- 1. 点击左侧导航栏中的 **“应用程序”**（Applications）标签页

- 2. 再点右上角的 **“连接后端”**（Connect Backend）按钮

- 3. 填写以下信息：

- • **名称**：OpenBB 本地后端

- • **URL**：`http://127.0.0.1:6900`

- 4. 点击 **测试连接**如果一切正常，你会看到提示：✅ 测试成功，发现了 X 个应用程序！

- 5. 最后点击 **添加**（Add）即可。

### 🎉 完成！
从现在开始，你的 Workspace 就连接上了本地 OpenBB 平台，可以：

- • 🖥️ 可视化操作金融数据分析

- • 🔗 使用后端 API 做扩展集成

- • ⚙️ 运行 LLM 助手自动生成研究报告

- • 📊 构建个人量化工作流系统

如果你是自媒体人、财经内容创作者、独立分析师，强烈建议试一试这个组合：

**本地 OpenBB 平台 + Workspace 可视化前端**，简直就是属于个人的“彭博终端工作站”！
