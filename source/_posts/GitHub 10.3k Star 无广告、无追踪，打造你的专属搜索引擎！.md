### Whoogle Search 的核心功能
1. **隐私保护，彻底避免数据泄露**Whoogle Search 致力于保护用户隐私，完全不收集个人信息。它不追踪 IP 地址，也不使用第三方 Cookie 或 JavaScript。这意味着你的搜索行为是完全私密的，外界无法通过你的搜索记录获取任何个人数据。

2. **无广告，专注搜索体验**Whoogle Search 的最大亮点之一就是它的无广告设计。告别了传统搜索引擎中的广告干扰，Whoogle Search 为用户提供干净整洁的搜索页面，让你能够专注于获取信息，而不是被各种广告打扰。

3. **高度定制，个性化设置**Whoogle Search 允许用户根据个人需求进行深度定制。你可以调整搜索结果的显示方式、选择主题模式、设置语言等，以打造一个完全符合你使用习惯的搜索引擎。无论是界面的简洁性，还是搜索结果的展示方式，都可以自由配置。

4. **增强隐私保护，支持代理和 Tor**如果你对隐私保护有更高的要求，Whoogle Search 还提供了通过代理或 Tor 网络进行搜索的功能。这意味着无论你处于哪个网络环境中，你的隐私都能得到最大程度的保护，完全避免了任何数据泄露的风险。

5. **快速部署，轻松上手**Whoogle Search 提供了简单高效的部署方式，无论你是使用 Docker 还是 Heroku，甚至手动安装，都能够快速搭建你的专属搜索引擎。即使你是技术小白，也能轻松上手。

### 如何安装 Whoogle Search？
**Windows 系统安装教程**- 1. 确保你的 Windows 系统已安装 Docker。

- 2. 打开终端，运行以下命令：`docker pull benbusby/whoogle-search
docker run --publish 5000:5000 --detach --name whoogle-search benbusby/whoogle-search:latest`

- 3. 在浏览器中访问 http://localhost:5000，即可开始使用 Whoogle Search。

**macOS 系统安装教程**- 1. 确保安装了 Docker。

- 2. 在终端中运行以下命令：```
`docker pull benbusby/whoogle-search
docker run --publish 5000:5000 --detach --name whoogle-search benbusby/whoogle-search:latest`
```

- 3. 打开浏览器，访问 http://localhost:5000，即可体验 Whoogle Search 的强大功能。

### 注意事项
- • **隐私设置**：虽然 Whoogle Search 已经为用户提供了强大的隐私保护，但为了进一步增强隐私，建议在部署时使用代理或 Tor 网络。

- • **定制化配置**：如果你想根据个人需求调整 Whoogle Search 的界面或功能，可以通过编辑 `whoogle.env` 文件来进行配置。

- • **部署环境**：建议使用 Docker 部署，因为它简单且高效，适合大多数用户。
