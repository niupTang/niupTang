如何使用Super Tiny Icons？

如果是前端开发通过npm安装，然后在你的项目中引用所需的图标。例如，如果你想使用GitHub的图标，你可以这样做：

`npm install --save super-tiny-icons`然后在你的代码中引用：

```
`import logo from "super-tiny-icons/images/svg/github.svg";

<img src={logo} />`
```
CSS定制

你还可以利用CSS来自定义图标的外观，比如改变大小、边框半径等，以适应不同的设计需求。
