# **StreamCap 是什么？**
**StreamCap** 是由开发者 **ihmily** 创建的一款开源屏幕捕获工具，支持：

- • **低延迟屏幕录制**（适用于直播、教程录制）

- • **RTMP推流**（可直接推送到B站、YouTube、Twitch等平台）

- • **AI增强功能**（如实时字幕生成、人脸追踪等）

- • **轻量化设计**（占用资源少，适合低配电脑）

# **核心功能**
### **1. 超低延迟屏幕捕获**
- • 支持 **DXGI、GDI** 等多种捕获方式，兼容Windows系统

- • 可自定义帧率、分辨率，平衡画质与性能

### **2. 一键RTMP推流**
- • 直接推流到 **B站、斗鱼、YouTube、Twitch** 等平台

- • 支持 **本地录制**（MP4、FLV格式）

### **3. AI增强模式（实验性）**
- • **实时语音转字幕**（可用于直播字幕）

- • **人脸/物体追踪**（自动调整画面焦点）

- • **背景虚化/替换**（类似Zoom虚拟背景）

### **4. 轻量化 & 可扩展**
- • 内存占用低，适合长时间直播

- • 提供 **HTTP API**，可集成自动化脚本
