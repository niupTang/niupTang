* 戳上方蓝字“**牛皮糖不吹牛**”关注我

       大家好，我是**牛皮糖**！今天给大家带来一款非常有趣的桌面美化工具——**MD3 Windows**。如果你跟我一样，喜欢**Material Design**风格，尤其是喜欢那种简洁、动态且高效的界面设计，那么这款工具绝对不容错过！它不仅让你的Windows桌面焕然一新，还支持自动从壁纸中提取颜色，打造一个美观的动态桌面，完全模仿了Android 12+的设计风格。

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrTfogcUcbkgeAzO8z51EL4MQrYd2PwQEDLTscRSIVkf9v3Wo6MR6CnWPXTdbOQTjadW91ZYTK3RCQ/640?wx_fmt=png&from=appmsg)

**MD3 Windows**是一个基于Rainmeter的皮肤，它带来了一些非常实用的功能，让你对桌面的掌控更加得心应手。**核心特点**包括：

- • **时钟**：清晰显示本地时间。

- • **日期**：准确显示当前日期。

- • **天气**：实时更新你所在位置的天气信息。

- • **音乐**：显示当前正在播放的音乐，专辑封面、歌曲、艺术家等信息。

- • **电池**：轻松查看电脑的电池电量及其它相关信息。

- • **文件夹快捷方式**：集成WiFi、蓝牙、下载、文档和图片等常用工具。

- • **活动提醒**：列出即将到来的节日活动和重要日期。

- • **问候信息**：每一天都会收到充满温暖的问候，带来好心情。

- • **监控面板**：实时显示RAM、CPU、硬盘以及SWAP内存的状态，保持你对系统性能的掌控。

- • **自定义图标包**：内置50+个图标，让你自由定制任务栏。

- • **任务栏、开始菜单、音量控制**：全方位的自定义UI，轻松替代传统的Windows UI，提升使用体验。

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrTfogcUcbkgeAzO8z51EL4MuK2HhXzaz4eLEawY4tzSNRUbYNYGnULwrx3vW7oevEiad7S7PPOhDNA/640?wx_fmt=png&from=appmsg)

### **功能对比：基础版与PRO版**
基础版和PRO版的区别主要体现在**样式和功能数量**上。例如，**时钟**的风格，基础版提供6种，而PRO版则多出4种；同样，**活动**和**天气**等功能的风格也有显著差异，PRO版拥有更多的选择。**PRO版**用户还将获得独享的自定义电源菜单和工具菜单等功能，非常适合有高级需求的用户。

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrTfogcUcbkgeAzO8z51EL4MGr4JUN4stlGkPOjNdua3uvHfbbyAytqJMicKxhicJY8ibsxMvL1A8tUTA/640?wx_fmt=png&from=appmsg)

### **安装步骤**
- 1. **安装/更新Rainmeter**：首先，确保你已经安装了适用于Windows的Rainmeter软件。

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrTfogcUcbkgeAzO8z51EL4MoDEhRqTKoyZMpdiaYUQbRLqcZjhhCQvUOHYR7tiaXAx8y2AGuQID8ohA/640?wx_fmt=png&from=appmsg)

- ![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrTfogcUcbkgeAzO8z51EL4MKJ3mTnXLCenak4V6qwyk3Fpp0GozaFuSf1iapFlUKS5iaJuo0n2OicWXg/640?wx_fmt=png&from=appmsg)

- 2. **下载MD3-Windows皮肤**：获取最新版本的MD3-Windows皮肤。

- 3. **安装皮肤**：双击下载的.rmskin文件，安装向导会自动打开，按照屏幕提示完成安装即可。如果安装过程中皮肤没有自动加载，可以通过Rainmeter皮肤管理器手动加载皮肤和布局。

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrTfogcUcbkgeAzO8z51EL4MtzdxGCpTWeluDBYS1NMan4o0LDCC60CN8qaEm3B2UHltkBmr3L0SNQ/640?wx_fmt=png&from=appmsg)

- ![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrTfogcUcbkgeAzO8z51EL4MsmQFTsNe2qVnC051qHNO0ZIR9HplMXZjT4MEoLfZ9wicXLmVpa4Q5xw/640?wx_fmt=png&from=appmsg)

### **温馨提示**
- • 由于插件限制，**MD3 Windows**仅在Windows的暗模式下才能正常工作，亮模式下会出现不兼容问题。

- • 不要尝试修改皮肤文件的尺寸，这可能会导致皮肤出现异常。

这款工具完全可以提升你的Windows桌面体验，让每次使用电脑都充满了惊喜。如果你喜欢这款**Material风格的桌面皮肤**，赶紧尝试一下吧！
