* 戳上方蓝字“**牛皮糖不吹牛**”关注我

    大家好，我是牛皮糖 ! 今天要介绍的是一个在GitHub上非常受欢迎的开源项目——Bilibili Evolved。这个项目拥有超过22.6k的Star，它是一个针对哔哩哔哩（Bilibili）的浏览器扩展，旨在为用户带来更加丰富和个性化的观看体验。

![](https://mmbiz.qpic.cn/mmbiz_jpg/p1ESIQQvfrSKxRPTlRYq29Khzaz0W9IoiaHlOM0UFRAJPLibnmQx8tjHuDuZb6wW3c21On8NnO4Df8LCHeU4MWxw/640?wx_fmt=jpeg&from=appmsg)

这个项目由开发者the1812发起，它不仅仅是一个简单的浏览器插件，而是一个功能强大的工具集，包含了多种增强功能，让用户在使用Bilibili时能够享受到更加舒适和便捷的服务。

### 核心功能
1. 视频播放增强：提供多种播放器设置选项，比如视频画质选择、自动播放、自动全屏等。

2. 界面个性化：允许用户自定义页面布局和样式，包括夜间模式、隐藏无关元素等。

3. 弹幕功能优化：增强弹幕的显示效果，提供屏蔽关键词、智能防挡字幕等功能。

4. 直播区块改进：为直播页面添加更多实用功能，如自动领取奖励、礼物自动感谢等。

5. 下载助手：支持视频、弹幕、字幕等内容的下载。

6. 多平台支持：不仅支持Chrome、Firefox等主流浏览器，还支持Microsoft Edge和Opera。

## 推荐配置

- 操作系统: Windows 10 / macOS Sonoma

- 分辨率: 3840x2160, 缩放 200%

- 浏览器: Chrome 120+ / Firefox 127+

- 处理器: Intel Core i7-11700 / AMD Ryzen 7 5800

- 内存: 32GB

- 脚本管理器: Tampermonkey 5.0 / Violentmonkey 2.18

- 显卡: GeForce RTX 3060 / Radeon RX 6600XT

- 网络: 20MB/s

Bilibili Evolved是一个值得每个Bilibili用户尝试的浏览器扩展。它通过提供丰富的功能和个性化设置，极大地提升了用户的观看体验。如果你也是Bilibili的忠实用户，不妨试试这个项目，相信它会给你的Bilibili之旅带来不一样的色彩！
