重磅开源 | Cobalt：最简洁优雅的媒体下载工具

### 技术分享 | Star破2万的全能下载工具，前后端完整开源
大家好，我是牛皮糖。在日常工作中，我们经常需要下载一些视频或音频资源，但市面上的下载工具要么广告满天飞，要么需要付费订阅。今天给大家推荐的这个开源项目 **Cobalt** 就很好地解决了这些痛点。

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrT2Ky3F5R81Y7EgicuILTE6xKNHVTK5M6eXpUNGyA9hFuBgWHG7ofIsMCoabILLCMPlia1xrHKDdvOg/640?wx_fmt=png&from=appmsg)

### 核心特色：
- • **简单直观**：复制链接，获取文件，就这么简单。

- • **零广告干扰**：无广告、无跟踪器、无付费墙。

- • **多平台支持**：覆盖YouTube、Twitter、Reddit、SoundCloud等主流平台。

- • **完全开源**：前后端代码全部开源，透明可靠。

### 技术栈：
- • **前端**：Svelte (47.3%)

- • **后端**：JavaScript (39.9%)

- • **类型支持**：TypeScript (12.2%)

### 实现原理：
Cobalt 的工作方式类似于高级代理，它不会缓存任何内容，仅用于下载公开可访问的内容。这保证了工具的合法性和安全性。

### 主要功能：
- • **多媒体下载**：支持音频和视频下载。

- • **社交平台支持**：包括YouTube、Twitter、Reddit、SoundCloud、TikTok、Vimeo、VK等。

### 项目架构：
项目采用 monorepo 结构，包含：

- • API 服务

- • Web 前端

- • 相关工具包

- • 完整文档

### 使用注意：
- • 仅用于下载免费且公开的内容。

- • 遵守相关平台的使用条款。

- • 用户需对下载内容负责。

Cobalt 的设计理念是让下载变得简单且不烦人。如果你也厌倦了各种下载工具的广告轰炸，不妨试试这个开源替代方案。
