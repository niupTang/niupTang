## 什么是 Copyparty？
Copyparty 是一个 **一脚本启动** 的 Python 文件服务器，支持可加速的断点续传、文件去重、媒体索引、自定义协议（如 WebDAV、FTP、TFTP）。开箱即用，同时适配现代浏览器和多种传输协议，

这意味着：你只需一条命令，就能打开一个功能完整的文件中转站，无需额外依赖，适用于各种平台 — 包括 Windows、Linux、macOS 甚至老设备

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrS1ZTxoIsvtjNkXewkI4Nz9eKZw53GXZEOqics0VjIlV3XlZk4X1BaAq0wSOPkTJaLTLTwgfSn1L9g/640?wx_fmt=png&from=appmsg)

## 核心功能亮点
- • **可中断续传 + 并行加速上传**：上传大文件再也不用从头开始

- • **文件去重（Dedup）**：上传重复文件时自动识别，节省空间与时间

- • **集成多协议服务**：不仅支持 HTTP，还提供 WebDAV、FTP、TFTP 等多种访问方式，兼容性极强。

- • **媒体资源管理**：自动生成缩略图、预览 Markdown 文件，还能在线播放音频，支持几乎无缝文件浏览体验。

- • **浏览器原生交互操作**：支持拖拽上传、文件搜索、在线编辑、ZIP 打包下载等 UI 功能，免去繁琐操作。

- • **自定义功能丰富**：支持上传生命周期管理（可设置文件自动删除）、上传撤回（unpost）、实时日志查看、搜索索引等。

## 社区反馈
- • r/DataHoarder 用户称其为“绝妙应用”，体积小但功能强大。

- • 有用户表示它比 Samba 或 SFTP 在速度与使用体验上更胜一筹，兼容多平台的浏览器访问也尤为出色，包括旧 iPad 或老 Mac(Reddit)。

- • 社区用户分享：它支持写入权限的分享、自带媒体均衡器、ZIP 打包下载和文件索引等，高度实用(Reddit)。

## 总结推荐理由
**Copyparty 是一款为高度自定义与高效文件管理而生的轻量服务器**，适合：

- • 快速搭建私有媒体库 

- • 在局域网内分享文件或收集资料

- • 替代速度缓慢的 SMB/NFS 访问方式

- • 控制上传生命周期或文件访问权限

- • 在各种设备上稳定运行

它不仅功能强大，而且界面简洁，适合折腾一族与高效办公需求。
