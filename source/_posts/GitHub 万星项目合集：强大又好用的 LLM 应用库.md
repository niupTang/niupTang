* 戳上方蓝字“**牛皮糖不吹牛**”关注我

    

大家好，我是牛皮糖！最近在研究 LLM（大语言模型）的实际应用，碰到了一个开源项目 **Awesome LLM Apps**，它将 LLM 与 **RAG（检索增强生成）** 和 **AI Agent（人工智能代理）** 完美结合，构建了许多实用又有趣的工具！

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrRq4iaYn7bJicYVOAXuf2XX969B68MRxONk9j1XV3GzFppxVmImLzIV7icdnLGGROtgvJV2XWxzDztHg/640?wx_fmt=png&from=appmsg)

**项目亮点**
这个项目不仅适合像我这样的技术开发者，也对对技术感兴趣的非程序员友好。它展示了 LLM 的多种实际应用方式，无论是本地部署的开源模型，还是对接 OpenAI、Anthropic 等主流平台的服务，都能找到完整的解决方案。

**核心功能**

- • **AI 智能代理**：帮助用户处理复杂任务，比如 AI 财务顾问、AI 健康规划、AI 招聘助手等。

- ![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrRq4iaYn7bJicYVOAXuf2XX96cr8xibP6ZGPlqBY7f7emGSTQbt5uHibUPUr81Y2jA348pANJWfhfQV9g/640?wx_fmt=png&from=appmsg)

- • **检索增强生成（RAG）**：让 LLM 更高效，比如结合本地数据库、混合搜索功能等。

- • **个性化记忆功能**：某些应用还能“记住”用户输入的内容，提供更贴心的服务。

- • **交互式聊天功能**：实现与代码库、文档、邮件等的智能对话。

**如何快速上手？**

- 1. **下载代码**`git clone https://github.com/Shubhamsaboo/awesome-llm-apps.git  `

- 2. **安装依赖**```
`cd awesome-llm-apps  
pip install -r requirements.txt  `
```

- 3. **运行项目**
每个项目都有详细的 README 指南，只需按步骤操作即可轻松部署！

**项目适合谁？**

- • **初学者**：想了解大语言模型和 AI 应用，学习 RAG 和 AI Agent 的实践方法。

- • **开发者**：快速开发自己的 LLM 应用，或从开源案例中获取灵感。

- • **企业团队**：结合开源项目快速打造行业解决方案，比如智能客服、健康诊断等。
