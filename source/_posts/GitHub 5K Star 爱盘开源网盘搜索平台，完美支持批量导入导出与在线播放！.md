### 部署方式
如果你也想体验这个开源项目，部署起来非常简单。你可以选择以下两种方式?

- 1. **Vercel 部署（推荐）**
点击项目页面上的 "Deploy with Vercel" 按钮，自动一键部署到 Vercel 平台，非常方便?

- 2. **Docker 部署（推荐）**
如果你更习惯使用 Docker 部署，爱盘也提供了完整的 Docker 部署文档。只需要执行以下命令即可启动服务：`cp .env.example .env
docker compose up -d`

### 开发指?
如果你打算参与开发，可以按照以下步骤进行?

- 1. **安装依赖**
使用 pnpm 包管理器安装项目依赖：`pnpm install`

- 2. **配置环境变量**
根据 `.env.example` 文件配置环境变量：```
`cp .env.example .env`
```

- 3. **数据库设?*
生成 Prisma 客户端并执行数据库迁移：```
`npx prisma generate
npx prisma migrate deploy`
```

- 4. **启动开发服务器**
启动本地开发服务器：```
`pnpm run dev`
```
