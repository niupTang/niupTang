* 戳上方蓝字“**牛皮糖不吹牛**”关注我

    大家好，我是牛皮糖！Deskreen 是一款非常实用的开源软件，是一款跨平台的无线投屏软件，它允许用户将电脑上的屏幕内容共享到其他设备上，如手机、平板甚至另一台电脑。这款软件的最大特点是使用简单、不挑设备，且安全性高。

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrRVqJnagibxTTY6t80UslQo9ic3hCctgGB3eOH5614Fwt8sHNKxYfOoeucF5YOoiaUZwkgF1JiaatvDog/640?wx_fmt=png&from=appmsg)

### Deskreen 特点
- • **跨平台支持**：Deskreen 支持 Windows、Mac、Linux，可以在这些操作系统上安装使用。

- • **无线投屏**：通过本地 Wifi 局域网实现投屏，无需额外硬件或连接线。

- • **设备兼容性**：接受投屏的设备没有限制，只要能使用浏览器访问网页即可，包括 iOS 和安卓设备。

- • **安全性**：提供密码保护功能，确保只有授权用户能投屏。

- • **多设备支持**：可以同时投屏到多个设备上。

- • **界面语言**：软件界面支持中文。

### 无线投屏的作用
- • **提高工作效率**：在需要多显示器的环境中，Deskreen 可以提供额外的显示空间，减少切换窗口的需要。

- • **会议和教学**：在会议或教学场景中，共享屏幕可以增加沟通效率。

### 使用 Deskreen 实现无线投屏的步骤
- 1. **下载和安装**：从 Deskreen 官网下载对应版本的软件包并安装。

- 2. **授权**：首次打开软件需要授权录制屏幕。

- 3. **连接设备**：确保电脑和接收共享的设备在同一个 Wifi 网络。

- 4. **生成二维码和网址**：打开 Deskreen 软件后，会自动生成一个二维码和网址。

- 5. **扫码或访问网址**：使用平板或手机扫码，或在浏览器里访问生成的网址。

- 6. **确认投屏**：在 Deskreen 确认投屏，选择投屏整个屏幕或某个窗口。

### 注意事项
- • Deskreen 软件支持中文，但首次打开时会有英文内容，点击绿色按钮跳过即可。

- • Deskreen 不能在接收设备上操作电脑，只能共享屏幕内容。

### 免费开源说明
Deskreen 是免费开源软件，源代码基于 AGPL-3.0 开源协议托管在 Github 上，任何人都可以免费下载使用。
