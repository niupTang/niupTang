## Open Lovable 能做什么？
### 从网站到 React App，只需一句话
Open Lovable 能 **抓取任意网站内容**，再通过 AI 分析并 **生成对应的 React 项目**，你只需运行本地服务器，就能拥有一个可定制的 React 应用。完全像魔法一样快！

### 核心技术栈组合：Firecrawl + Kimi + E2B Sandbox
- • **Firecrawl**：顶级爬虫 API，支持 JavaScript 渲染页面，抓取网页结构和内容

- • **Kimi AI**：高性能语言模型，快速分析网页结构并生成 React 组件；

- • **E2B Sandbox**：安全沙箱环境，防止 AI 生成代码带来的安全隐患

### 完全开源 + 高度可控
MIT 协议开源，你可以本地部署、定制修改、避免厂商锁定。只需为你使用的 AI 服务付费，没有额外门槛。

## 如何快速上手？
- 

- 1. 克隆项目并安装依赖：`git clone https://github.com/mendableai/open-lovable.git
cd open-lovable
npm install`

- 2. 创建 `.env.local`，填入必要的 API Key：

- • 必填：

- • `E2B_API_KEY`：执行沙箱

- • `FIRECRAWL_API_KEY`：网站抓取

- • 可选（至少填一个）：

- • `ANTHROPIC_API_KEY` / `OPENAI_API_KEY` / `GEMINI_API_KEY` / `GROQ_API_KEY`

- 3. 启动服务并访问：```
`npm run dev`
```
打开 `http://localhost:3000`，输入任意网站 URL，即可生成 React 克隆项目

- 当然如果你和我一样想偷懒部署我建议直接使用 AI 编程工具一键部署。

- 

- 简单的要他做一GitHub 热榜的页面，1 分钟内生成了对应的前端页面。
