    这个项目，叫做Deep-Live-Cam，无需专用硬件加速。

Deep-Live-Cam 是一个实时人脸交换和一键视频深伪工具。它能够利用单张图像对视频或其他图像进行人脸替换，你可以用它把一个人的脸换到另一张图片或视频上，适合用于视频制作、动画创作等场景。效果贼好，还被开源——项目一下就火了。直奔GitHub热榜第一，24小时狂澜1600+星。

### 安装方法
  支持 CPU 和 GPU 加速，但 CPU 版本运行速度可能较慢。

- 需要 Python 3.10（推荐）、pip、git、ffmpeg 和 Visual Studio 2022 运行时（仅限 Windows）。

- 通过 git 克隆仓库，并下载必要的模型文件放入 "models" 文件夹。

- 使用 pip 安装依赖项，并可以通过 `python run.py` 命令运行程序。

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrRCrkhsgTMvbMxuVBk4fn8VH02ibtcWzPBxypicc4y765icL67kKib7ibqM7rOnLoCzXN6NgVhZfKEfNIA/640?wx_fmt=png&from=appmsg)

### 使用方法
- 首次运行程序时会下载一些模型，大小约为 300MB。

- 用户需要选择一张包含所需面部的图片和目标图片或视频，然后点击 "Start"。

- 处理完成后，会在用户选择的输出目录中创建一个名为 `<video_title>` 的目录，其中可以看到实时交换的帧，并最终生成输出文件。

- 
`注意：首次运行该程序时，它将下载一些大小约为 300MB 的模型。`
