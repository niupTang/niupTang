* 戳上方蓝字“**牛皮糖不吹牛**”关注我

    大家好，我是牛皮糖！最近我在 GitHub 上发现了一个开源项目——**FinRobot**，它是一个专为金融分析设计的 AI 代理平台，利用大型语言模型（LLMs）进行深度分析。

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrQufXNBmmt6s56acx5XScKePl2yhiaFA29HDzskHY8WF3WZsQlkarq7asdHFgNLB1QjqsU0Zzyibicgw/640?wx_fmt=png&from=appmsg)

**FinRobot 的主要功能：**

- • **金融 AI 代理层**：通过金融链式思维（CoT）提示，增强复杂分析和决策能力。市场预测代理、文档分析代理和交易策略代理利用 CoT 将金融挑战分解为逻辑步骤，提供精确、可操作的洞察。

- • **金融 LLM 算法层**：配置和利用专门调整的模型，针对特定领域和全球市场分析。

- • **LLMOps 和 DataOps 层**：实施多源集成策略，为特定金融任务选择最合适的 LLM，利用一系列先进模型。

- • **多源 LLM 基础模型层**：支持各种通用和专业 LLM 的即插即用功能。

- ![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrQufXNBmmt6s56acx5XScKeyQgao1Oh4gbC7wXSUY9Kp42SJljs8S2o0x6zTn7XMKdszia9VzkCbmA/640?wx_fmt=png&from=appmsg)

**使用教程：**

- 1. **创建虚拟环境：**`conda create --name finrobot python=3.10
conda activate finrobot`

- 2. **克隆仓库：**```
`git clone https://github.com/AI4Finance-Foundation/FinRobot.git
cd FinRobot`
```

- 3. **安装依赖：**```
`pip install -U finrobot`
```

- 4. **配置 API 密钥：**

- • 将 `OAI_CONFIG_LIST_sample` 重命名为 `OAI_CONFIG_LIST`，并添加您的 OpenAI API 密钥。

- • 将 `config_api_keys_sample` 重命名为 `config_api_keys`，并添加您的 Finnhub、FMP 和 SEC API 密钥。

- 5. **探索教程：**导航到 `tutorials_beginner` 或 `tutorials_advanced` 目录，查找入门和进阶教程。

**注意事项：**

- • **优点：** FinRobot 提供了一个全面的框架，集成了多种 AI 技术，满足金融行业多样化的需求。

- • **缺点：** 由于是开源项目，可能需要一定的技术背景才能充分利用其功能。

**总结：**

FinRobot 是一个功能强大的金融 AI 代理平台，适用于市场预测、报告生成和数据分析等多种金融应用。通过开源社区的支持，用户可以根据自身需求进行定制和扩展。
