* 戳上方蓝字“**牛皮糖不吹牛**”关注我

    大家好，我是牛皮糖！ 今天在微信读书看到周志明的《凤凰架构》然后还有对应的项目在 GitHub 开源了Spring Boot、Spring Cloud 、Kubemtes 、 Istio 和AWS Lambda 五种架构风格的工程。    

    凤凰架构主要是深入剖析分布式架构，详细讲解远处服务、事务处理、透明多级分流系统、系统安全等概念。架构并不是“发明”出来的，是持续进化的结果。
