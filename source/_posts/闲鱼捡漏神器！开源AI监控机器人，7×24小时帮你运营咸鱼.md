# 项目简介
**AI Goofish Monitor** 是 dingyufei615 开发的一款基于 **Playwright + AI** 的闲鱼商品实时监控机器人。它支持多任务并发监控、AI 智能筛选商品并通过 WebUI 管理任务，实时推送满足需求的优质商品。

## 📌 核心功能亮点
- • **🧠 AI 驱动任务创建**：用自然语言描述需求，AI 自动生成筛选规则，如“筛选二手手机，卖家信誉高”。

- • **⏱ 多任务并发监控**：多个关键词同时监控，互不干扰。

- • **📊 实时商品分析**：AI 分析商品描述、图片、卖家信息，迅速判断是否符合要求。

- • **🌐 可视化 Web 管理界面**：支持任务管理、日志查看、筛选规则编辑，完全无命令行操作。

- • **🚨 即时通知推送**：支持 ntfy.sh 或企业微信机器人推送符合条件的商品信息。

- • **⚙ 自定义筛选**：每个任务可单独设置关键词、价格区间、AI 分析 prompt 等。

- • **🧊 无头模式 & 链接转换**：可无头运行，自动将 PC 链接转为手机端链接，省资源也易于操作。

##  安装与使用流程（总结）
- 1. 克隆仓库并进入目录：`git clone https://github.com/dingyufei615/ai-goofish-monitor
cd ai-goofish-monitor`

- 2. 安装依赖：```
`pip install -r requirements.txt`
```

- 3. 创建 `.env` 配置文件，填写 OpenAI Key、通知服务 URL、运行模式配置等。

- 4. 运行 `python login.py` 获取闲鱼登录状态（扫码登录生成会话状态文件）。

- 5. 编辑 `config.json` 添加任务配置，例如关键词、价格范围、AI prompt。

##  使用建议
- • **务必遵守**闲鱼网站协议与 `robots.txt`，避免频繁请求导致限制。

- • **注意筛选规则调整**：有用户反馈希望增加按发布时间、价格排序功能，目前已有 feature request 在等待实现中。

- • **AI 筛选模型兼容性**：使用 Gemini API 需正确配置 URL，否则可能报错。

- • **运行环境与反馈**：项目社区非常活跃，若遇到问题可查询 issue 区获取帮助。
