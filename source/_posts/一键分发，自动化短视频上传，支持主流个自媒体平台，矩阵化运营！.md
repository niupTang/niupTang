## 🚀 它能干嘛？
想象一下👇

你精心剪了一个 30 秒爆款短视频，还要挨个打开：

- • 抖音✅

- • 快手 ✅

- • 视频号 ✅

- • 百家号 ✅

- • B 站 ✅

- • ✅

... 每个平台点一点、传一遍、写文案、选标签、调整格式，上传完人都快崩溃了……

**这个项目的作者 dreammis 直接干了件大事：**

把所有主流社交平台上传接口自动化！ 你只需要一条命令，它帮你发遍全网！

## 🧰 怎么用？
### ✅ 本地运行（适合开发者）
[从零开始学Python-ANACONDA安装](https://mp.weixin.qq.com/s?__biz=MzkyNDYyODg0MQ==&mid=2247484877&idx=1&sn=4b6e1a228937b0e494eb257f78036e0a&scene=21#wechat_redirect)
`git clone https://github.com/dreammis/social-auto-upload.git
cd social-auto-upload

conda create -n social-auto-upload python=3.10
conda activate social-auto-upload

pip install -r requirements.txt
python main.py`

playwright install chromium firefox将需要的从 项目中`examples目录`中的`get_douyin_cookie.py` 、`upload_video_to_douyin.py` ，拷贝到当前根目录（如果你会设置环境变量的话，那么你可以不拷贝）

## 🧩 用法场景
- • 🧑‍🎨 **自媒体博主**：剪完视频直接 push 发全网！

- • 🏢 **品牌方/MCN**：统一发布多账号内容，效率拉满

- • 🧑‍💻 **开发者**：集成到自动内容生成系统，打造内容机器人

- • 😎 **自由职业者**：一个人也能干整个运营团队的事

## 📌 牛皮糖的建议
这个项目目前还在不断完善中，部分平台登录可能有反爬验证，但逻辑非常清晰，适合：

- • 改成自己团队的内部运营工具

- • 集成 GPT + Midjourney，自动生成内容 + 自动发平台

- • 做成 Web 控制面板，更适合小白使用
