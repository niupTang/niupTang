## 项目是什么？解决了什么痛点？
**GenAI Toolbox for Databases** 是一个基于 **MCP（Model Context Protocol）** 的开源服务器工具，专为 AI Agent 与数据库之间的安全、高效对接打造。

它简化了传统模型与数据库交互的繁琐流程——比如连接池、认证、安全授权、监控等都由它统一处理。

借助它，开发者无需重复造轮子，就能让智能体以自然语言直接对 SQL 数据库进行操作，比如敲 SQL、创建索引、生成代码，甚至进行表结构理解查询 。

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrQyrVpvM7oCCnhrqDCkKaf3QAXVQFMibZ3RjkDe9p6eHppRu53aHibNAbLiaNicnPianNle78McOiaKO5IQ/640?wx_fmt=png&from=appmsg)

## 核心亮点一网打尽
功能
说明
**MCP 协议支持**标准协议接口，实现 Agent 与数据库解耦通讯
**支持多种数据库**包括 PostgreSQL、MySQL、Spanner、AlloyDB、Cloud SQL 等
**集成监控与安全机制**支持 OpenTelemetry 监控、OAuth2/OIDC 认证、连接池等生产级能力
**快速工具定义**通过 YAML 配置即可定义数据库工具，便于 Agent 调用复用
**多语言 SDK**完整支持 Go、Python、JavaScript 客户端调用
**轻松部署**支持本地快速启动、Docker、Cloud Run 等多种部署方式
