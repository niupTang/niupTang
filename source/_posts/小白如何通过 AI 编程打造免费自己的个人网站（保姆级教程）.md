注册一个GitHub 账号，https://github.com/

3、 配置 GitHub 将本地代码上传到 Github 

对于 AI 编程工具说将代码上传到你自己的仓库， 这个仓库上面，如果没有下载 git 要 AI 编程工具给你下载，下载完后自己通过命令配置 git.

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrQb1uJibibpFxeYia3cK3jmEOIPSTibMJ6LP9aTh5sa8HTxrA4Ox0AA892KiaPmcR2EoZZqJ9CPLntvmsg/640?wx_fmt=png&from=appmsg)

对 AI 工具说 配置 git 将账号和仓库地址给他，然后最后自己输入密码。

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrQb1uJibibpFxeYia3cK3jmEOI9Ad9SXOgwniceujMv0prEGpQDpyKcdCP2QJfSfoDvm70PtRAsQm4BNg/640?wx_fmt=png&from=appmsg)

 配置完成后将代码上传到 GitHub 仓库 ，配置服务器。

4、 配置服务https://vercel.com

我们可以免费用 vercel 选择用GitHub 授权，配置对应的GitHub 仓库，只需要将仓库既可完成。

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrQb1uJibibpFxeYia3cK3jmEOISF6sJPQ5XUibZgvoLBvpiajtPm2RqDpqib9zwn9p4HlwBxW6Xibr6dqfJw/640?wx_fmt=png&from=appmsg)

打开项目，既可看到自己的域名服务，通过这个域名就能访问你自己个人网站了完全免费。
