    大家好，我是牛皮糖！自托管的聊天机器人构建器。该项目通过可视化的拖拽界面，让用户能够轻松创建高级聊天机器人，并将其嵌入网站中。它提供了 30 多种聊天构建块，支持自托管、分析工具、自定义域名和品牌定制等功能，适用于在线客服和销售支持等场景。

Typebot 可轻松创建高级聊天机器人。它提供了可适应任何业务用例的构建块。我会定期通过错误修复、新功能和性能改进来改进 Typebot。

聊天构建器具有 34 多个构建块，例如：

- 💬气泡：文本、图像/ GIF、视频、音频、嵌入。

- 🔤 输入：文本、电子邮件、电话号码、按钮、图片选择、日期选择器、付款（Stripe）、文件选择器......输入

- 🧠 逻辑：条件分支、URL 重定向、脚本 (Javascript)、A/B 测试

- 🔌 集成：Webhook/HTTP 请求、OpenAI、Google Sheets、Google Analytics、Meta Pixel、Zapier、Make.com、Chawoot，还有更多……

为您的聊天机器人设置主题以匹配您的品牌标识：

- 🎨 自定义字体、背景、颜色、圆度、阴影等

- 💪 使用自定义 CSS 实现高级主题。

- 💾 可重复使用的主题模板

在任何地方分享你的 typebot：

- 🔗 自定义域名

- 👨‍💻 使用本机 JS 库轻松嵌入为容器、弹出窗口或聊天气泡。

- ⚡ 超快嵌入库。无 iframe、无外部依赖、无性能影响。

- 💻 可通过 HTTP 请求执行

收集您的结果并获取见解：

- 📊 深入分析流失率、完成率等

- 📥 将结果导出为 CSV

专为开发人员打造：

- 🔓 无供应商锁定。功能构建时充分考虑了灵活性。
