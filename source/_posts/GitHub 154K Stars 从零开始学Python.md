* 戳上方蓝字“**牛皮糖不吹牛**”关注我

    大家好，我是牛皮糖！学习最大的难点就是不知道如何规划，每天学多少怎么学，这个开源项目从 0 开始 100 天学 Python 简直不要太香每天的规划，学多少都有规划，非常适合刚刚入学大学生和想要自学的同学。

目前，数据分析和数据挖掘是非常热门的方向，因为不管是互联网行业还是传统行业都已经积累了大量的数据，各行各业都需要数据分析师从已有的数据中发现更多的商业价值，从而为企业的决策提供数据的支撑，这就是所谓的数据驱动决策。
