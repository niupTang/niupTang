使用视觉模型将 PDF 转换为 Markdown。该项目基于视觉模型 API 服务，提供了将 PDF 文档转化为 Markdown 的功能。

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrQcT28MqwibGFE7zFj0Hm1e6yYQ0TicK4h0wmQCReaGvOKQfVOlHAjev0up4tofia8Nh6gJibX2Zt8fRg/640?wx_fmt=png&from=appmsg)

Zerox 是由 OmniAI 开发的一款开源 OCR 工具，旨在将文档（PDF、DOCX、图片）通过 AI 视觉模型转换为结构化的 Markdown 格式。它使用 GPT-4o-mini 实现高精度的文本提取和格式化，非常适合开发者、研究人员和内容创作者。

⚙️ **核心工作流程**

- • **输入转换**：支持 PDF、DOCX、PNG、JPG 等格式。文件被拆分为单页并转换为图像。

- • **AI 驱动的 OCR**：利用 GPT-4o-mini 分析图像，执行 OCR 并提取文本，同时理解语义（如保留表格、代码块）。

- • **Markdown 聚合**：将处理后的页面合并为一个 Markdown 文档，保持原始结构和格式。

✨ **关键特性**

- • **高精度**：在多样化数据集（手写文本、多语言内容、复杂布局）上训练。

- • **隐私优先**：本地处理文件，无需云上传，适合敏感数据。

- • **批量处理**：支持多文件或大型文档的并发处理。

- • **格式保留**：在 Markdown 输出中保持表格、列表和标题。

- • **多 SDK 支持**：支持 Python 和 Node.js 集成，便于实现。

🚀 **使用场景**

- • **研究**：从扫描的论文或报告中提取文本进行分析。

- • **内容创作**：将传统文档转换为可编辑的 Markdown，用于博客或维基。

- • **自动化**：集成到批量文档处理流程中（如法律或学术工作流）。
