* 戳上方蓝字“**牛皮糖不吹牛**”关注我

    大家好，我是牛皮糖！

    默认琥珀色 与 默认绿色 的对比，让人不禁想起那些年我们熟悉的计算机界面。今天，我们将探索一款名为 cool-retro-term 的终端仿真器，它不仅重现了旧阴极射线管屏幕的经典外观，还融入了现代技术的便捷与高效。

![](https://mmbiz.qpic.cn/mmbiz_jpg/p1ESIQQvfrRVhRibnqS8bkic9rIgeObH9xXhdKQWc5aU4LMtY5AE28ywZcK3BwvB5xDFb8RPwicaryceNibT7cLO1A/640?wx_fmt=jpeg&from=appmsg)

简介cool-retro-term 是一款终端仿真器，它模仿了旧式阴极射线管屏幕的外观和感觉。这款软件以其赏心悦目的界面、高度的可定制性以及轻量级的设计赢得了用户的喜爱。

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrRVhRibnqS8bkic9rIgeObH9x6sScm30j44sbSh7AgfZkvHmOZmJRRdD8G9urlMh1bTTU8m3r3SLNHQ/640?wx_fmt=png&from=appmsg)

技术基础cool-retro-term 使用了 qtermwidget（Konsole的QML端口），这是一个基于Qt5的QML组件，专为现代操作系统设计。你可以通过以下链接了解更多关于qtermwidget的信息：qmltermwidget GitHub。

系统要求- 操作系统：Linux 和 macOS

- 依赖：需要Qt5。推荐使用最新的LTS（长期支持）版本。

安装指南- 最新版本：访问 cool-retro-term 的发布页面，下载最新的 AppImage（适用于Linux）或 dmg（适用于macOS）文件。

- 官方存储库：大多数Linux发行版（如Ubuntu、Fedora或Arch）已经在其官方存储库中提供了 cool-retro-term。

自定义设置通过 cool-retro-term 的上下文菜单，你可以轻松访问并自定义颜色、字体和效果等设置，以满足你的个性化需求

构建指南如果你对如何构建 cool-retro-term 感兴趣，可以查看其 GitHub 页面上的wiki，那里提供了详细的构建指南，包括如何在Linux和macOS上进行构建。

项目地址：

**https://github.com/Swordfish90/cool-retro-term**
