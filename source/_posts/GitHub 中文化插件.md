* 戳上方蓝字“**牛皮糖不吹牛**”关注我

    大家好，我是牛皮糖！ 对于大家访问Github 因为是英文版本的所以会有一些阅读困难，其实GitHub 上面有一个开源的插件介绍如何安装中文化插件。

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrT3eDtflw9wnzeSsyQjJEUcnTIUkiaXO97cWiaOia5KoibfHSG7eSsKzS1SKmFeia9UBb4pzccUKBHnYQA/640?wx_fmt=png&from=appmsg)

- 中文化 GitHub 菜单栏，标题，按钮等公共组件

- 保留、完善正常功能

- 支持对“项目描述”进行人机翻译

**安装**

首先需要安装篡改猴浏览器此插件

https://www.tampermonkey.net/index.php

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrT3eDtflw9wnzeSsyQjJEUclib5h68rUCZiaIxHvhpSas2vTd1AHSfgMR0Hf9icojaWV3N66U61wsuMA/640?wx_fmt=png&from=appmsg)

****

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrT3eDtflw9wnzeSsyQjJEUcx0ulsibwYRsfaN4u7r0rjia9wAnnFBLFAGQYKsG8nqkVTx7jRBWx5Dog/640?wx_fmt=png&from=appmsg)

****

然后再进入下面这个脚本链接安装猴油脚本就可以了

https://github.com/maboloshi/github-chinese/raw/gh-pages/main.user.js

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrT3eDtflw9wnzeSsyQjJEUcwRkWwTqB1SIPQwDAk4aU9eibwut0Rajjxicrs229FLDbibBIlza0W1MVg/640?wx_fmt=png&from=appmsg)

****

其实油猴脚本还有很多，被称为程序员的天堂，有很多插件都可以再这上面去找到。除了油猴还有其他的脚本管理器也可以下载安装。

浏览器脚本管理器Chrome 或 基于 Chromium 内核的浏览器Tampermonkey 或 ViolentmonkeySafari 浏览器（macOS, iOS, iPadOS）Macaque 或 StayFirefox 或 基于 Gecko 内核的浏览器Tampermonkey 或 ViolentmonkeyVia 浏览器（Android）浏览器内自带****
