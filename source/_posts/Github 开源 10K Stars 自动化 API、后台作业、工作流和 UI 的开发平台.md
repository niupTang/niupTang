    大家好，我是牛皮糖！今天我要为大家介绍一个非常强大的开源项目——Windmill。Windmill是一个为开发者提供的基础设施平台，用于构建内部工具（APIs、后台任务、工作流和UIs）。它可以看作是Retool、Pipedream、Superblocks的自托管替代品，同时也是一个简化的Temporal，具有自动生成的UI和自定义UI，可以触发工作流和脚本作为内部应用程序。

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrQl53jez91PKjicDI332d32bUlN4VcePNrmNzR63YSpwIwfy3lgUyCOnaibdiasGjtnKn62iaBGkOdPMg/640?wx_fmt=png&from=appmsg)

**通过脚本实现**

用 Python、TypeScript、Go 或 Bash 定义一个最小且通用的脚本来解决特定任务。

Windmill允许你将脚本转换为可共享的UI，并且可以将这些脚本组合成流程，或者用于构建更丰富的低代码应用程序。支持的脚本语言包括Python、TypeScript、Go、Bash、SQL和GraphQL。这意味着，无论是数据处理、自动化任务还是构建复杂的后台逻辑，Windmill都能提供强大的支持。
