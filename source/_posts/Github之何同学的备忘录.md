* 戳上方蓝字“**牛皮糖不吹牛**”关注我

    大家好，我是牛皮糖！这两天看到何同学风波，我来蹭一点热度，我去看了GitHub 上开源的“何同学项目”ASCII 生成器（图像转文本、图像转图像、视频转视频）。

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrTrOTAnkuz0D8Qz7ndiaxjXPfVaEb4OjXvsYmmMjQAibHlNJZ9Hup8ibJqzkKJHX21TdhqqFAtiaiaCyRA/640?wx_fmt=png&from=appmsg)

ASCII 生成器的核心功能是将给定的输入图像转换成不同语言的文本格式存储的ASCII艺术（.txt）。这意味着你可以选择你喜欢的任何语言，无论是英语、德语、法语、韩语、中文还是日语，ASCII 生成器都能帮你实现。

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrTrOTAnkuz0D8Qz7ndiaxjXPXBu5xCQBXicicVpey6piaPibA41YEx62KaDM2nQeV5ryzpfqJFV3iaar1hg/640?wx_fmt=png&from=appmsg)

![](https://mmbiz.qpic.cn/mmbiz_jpg/p1ESIQQvfrTrOTAnkuz0D8Qz7ndiaxjXPQiaojyp6k8waN4PwbwcGXowFZcMaSaHDyCbpQEdqDUXdNnxq6vEKJWw/640?wx_fmt=jpeg&from=appmsg)

不仅如此，ASCII艺术生成器还支持将输入图像生成为不同语言的图像格式（.png、.jpg等），并且提供了两种背景选项：黑色背景配白色字符，或者白色背景配黑色字符。这种灵活性让你可以根据不同的应用场景和个人喜好来选择最合适的输出格式。

对于视频爱好者来说，ASCII艺术生成器同样提供了强大的功能。你可以将输入视频转换成不同语言的视频格式（.avi，.mp4等），并且可以选择灰度或彩色格式的输出。这为视频内容创作者提供了一个全新的视角和创作工具。

**环境要求**

- python 3.6

- cv2

- PIL

- numpy
