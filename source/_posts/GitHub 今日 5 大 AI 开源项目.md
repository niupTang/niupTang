大家好，我是牛皮糖！每天 GitHub 上都会有众多有趣且实用的项目崭露头角，下面就为大家介绍今天 GitHub 热榜上的热门项目。

## 1. rasbt / LLMs-from-scratch
- • **项目链接**：[https://github.com/rasbt/LLMs - from - scratch](https://github.com/rasbt/LLMs - from - scratch)

- • **项目介绍**：该项目可以帮助开发者一步一步用 PyTorch 从零实现一个类似 ChatGPT 的大语言模型。对于想要深入了解大语言模型原理和实现的开发者来说，这是一个绝佳的学习资源。目前拥有 71,100 颗星，今日新增 655 颗星。

- ![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrQMkqU2Nao1fVUYOmqUvcG8E4rjUg4I70cX05sAcGGwQQDShJDHdebicqyYPIV5pHVCPOpz1UBB32Q/640?wx_fmt=png&from=appmsg)

## 2. microsoft / markitdown
- • **项目链接**：https://github.com/microsoft/markitdown

- • **项目介绍**：这是一个 Python 工具，用于将文件和办公文档转换为 Markdown。对于经常需要处理文档格式转换的人来说，这个工具非常实用。它已有 74,189 颗星，今日新增 702 颗星。

- ![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrQMkqU2Nao1fVUYOmqUvcG8c3fVPYxm7Nw0ia3xpo2glLhgZDQkD8SczYHpZ7Ct2onXt72rV4IRF9g/640?wx_fmt=png&from=appmsg)

## 3. PowerShell / PowerShell
- • **项目链接**：https://github.com/PowerShell/PowerShell

- • **项目介绍**：PowerShell 是适用于每个系统的脚本语言和自动化框架。它可以在多种操作系统上运行，为系统管理和自动化任务提供了强大的支持。目前项目有 49,709 颗星，今日新增 486 颗星。

- ![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrQMkqU2Nao1fVUYOmqUvcG83KEW00JjrQvufAx0fF03ESpncr0HJHYsdWicT3KozicNp3EGh4BHahcA/640?wx_fmt=png&from=appmsg)

## 4. x1xhlol / system-prompts-and-models-of-ai-tools
- • **项目链接**：https://github.com/x1xhlol/system-prompts-and-models-of-ai-tools

- • **项目介绍**：该项目收集了 Cursor、Manus 等多种 AI 工具的系统提示、工具和 AI 模型。对于 AI 开发者和研究者来说，这是一个非常有价值的资源库。拥有 85,041 颗星，今日新增 685 颗星。

- ![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrQMkqU2Nao1fVUYOmqUvcG8AVY85Kp5gHApRAvL6XYIgnNzU2t7aa4d7FBajMjTdKicB4KOMX82MMQ/640?wx_fmt=png&from=appmsg)

## 5. Zie619 / n8n - workflows
- • **项目链接**：[https://github.com/Zie619/n8n - workflows](https://github.com/Zie619/n8n - workflows)

- • **项目介绍**：这里收集了 n8n 的所有工作流。n8n 是一个开源的工作流自动化工具，这个项目可以帮助用户更好地使用 n8n 来实现各种自动化任务。目前有 30,164 颗星，今日新增 498 颗星。

- ![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrQMkqU2Nao1fVUYOmqUvcG8uEpab4tfqtX18RbeP51icQpuWBv1duDia14vU7wkyc2Op2OIpYgtkVdw/640?wx_fmt=png&from=appmsg)
