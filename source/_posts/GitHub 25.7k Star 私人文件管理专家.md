* 戳上方蓝字“**牛皮糖不吹牛**”关注我

    大家好，我是牛皮糖！在这个信息爆炸的时代，我们每天都在产生大量的文件，如何高效地管理和访问这些文件成了一个让人头疼的问题。今天，我要给大家介绍一个在GitHub上拥有25.7k Star的开源项目——**filebrowser**。

![](https://mmbiz.qpic.cn/mmbiz_gif/p1ESIQQvfrQVMj0jib9M9kus7DSn1PzEOicNp3Kqdn7u3YkTx4icaUC5wUJicSOfYk5nt7gI8ZYBhT85toPJ9gp9vA/640?wx_fmt=gif&from=appmsg)

****

   filebrowser是一个开源的文件浏览器，它可以让你轻松地通过Web界面访问和管理文件。它支持多种操作系统，包括Linux、macOS和Windows，这意味着无论你使用哪种系统，都可以使用它来管理你的文件。

**核心功能：**

**
**

- 多平台支持：无论你是在个人电脑还是服务器上，都可以轻松部署。

- 简洁的界面：提供直观的文件浏览和操作界面，无需复杂的学习过程。

- 安全性：支持基本认证，保护你的文件不被未授权访问。

- 插件系统：允许你通过插件扩展功能，满足更多个性化需求。

**
**

**Docker 安装**

`docker run \
    -v /path/to/root:/srv \
    -v /path/filebrowser.db:/database.db \
    -v /path/.filebrowser.json:/.filebrowser.json \
    -u $(id -u):$(id -g) \
    -p 8080:80 \
    filebrowser/filebrowser`默认的账号密码

- 用户名:admin

- 密码：admin
