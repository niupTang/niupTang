十月期间，GitHub 新上榜了 122 个项目，其中，我们为您精选了九个旨在提升生产力的开源工具。这些工具完全免费且开源。

以下是这九个项目以及它们的关键特性：

### 1. EVWAR（桌面大模型工具）
EVWAR 的核心在于将 AI 大模型直接搬到了您的桌面上，解决了每次使用时需要打开大模型网站或进行截屏提问的痛点。

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrQAIibsNxLQiaJQWqibhMQ26UKKDTHFdX7VNDZTtZCv7SibQqa7sDCdMfsia8wzfhIrapgp6TKsWiagXheA/640?wx_fmt=png&from=appmsg)

- • **使用方式：** 下载后直接打开，通过按下 `Ctrl + Shift + E` 即可唤醒。

- • **功能：** 您可以直接对大模型提问，例如选择当前窗口并提问“请帮我总结一下当前页面”，即可获得结果。

- • **兼容性：** 它支持八大主流 AI 模型，无论您是 OpenAI 的老用户、Claude 的铁粉，还是本地部署的 Ollama 大模型，都可以随时随地使用。

### 2. Handy（离线语音转文字工具）
Handy 是一款完全离线工作的语音转文字工具，解决了三大常见痛点：

- • **隐私性：** 您的声音绝对隐私，只保留在您的电脑中，绝不会上传至云端。

- • **费用：** 永久免费，作为开源软件，没有付费墙。

- • **多模型支持：** 它支持多种模型以适应不同需求。在设置界面中，您可以选择 **Whisper small** 来追求速度，或选择 **Whisper large** 来确保准确性；来源推荐使用中等规格的模型。

- • **使用方式：** 按住 `Ctrl + Space` 说话，松开后文字就会出现在您的文档里，全程零网络、零延迟、零隐私顾虑。

- ![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrQAIibsNxLQiaJQWqibhMQ26UK7lW1UmVPSSIK4otsribCrQ3XjmB7S5CR6J9LP2E7hlN6G6icPBpMgIiaw/640?wx_fmt=png&from=appmsg)

### 3. YT Downloader（下载界“瑞士军刀”）
YT Downloader 被誉为下载界的“瑞士军刀”。

- • **功能强大：** 它不仅能下载 YouTube 视频，还能横扫抖音、推特、B站等平台的视频内容。

- • **使用简便：** 界面简洁，只需复制视频链接，在工具中按 `Ctrl + V`，视频的下载地址就会出现。

- • **性质：** 完全免费、绿色开源。

- ![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrQAIibsNxLQiaJQWqibhMQ26UKCPTkPtaekYkSVaeXugaodlTsfGjsYLNNfNC0xV5sKw2H9BJKdO9hRQ/640?wx_fmt=png&from=appmsg)

### 4. Media Downloader（下载界“终极缝合怪”）
如果说 YT Downloader 是瑞士军刀，那么 Media Downloader 就是下载界的“终极缝合怪”。

- • **集成性：** 它将市面上所有主流的下载工具都打包集成到了同一个图形界面中。

- • **架构：** 自带了一个工具包扩展系统，将常用的下载引擎（包括 YT Downloader）都集成在了一起。

- • **批量下载：** 它支持批量下载，例如您可以直接添加多个 B站视频地址，然后点击下载，所有资源都会同步保存到您的电脑上。

### 5. Mastry（自动化策略游戏）
### ![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrQAIibsNxLQiaJQWqibhMQ26UKEhB8RlbiaSEyYtYCiaf6icdgB5XRkcRNFLDfj2bCPDXCaR6XBW0ibI3N0A/640?wx_fmt=png&from=appmsg)

这是一个开源的策略游戏，结合了自动化流水线建造与塔防风格。

- • **特点：** 它完美地结合了塔防的爽快感和自动化的生产流程。

- • **技术细节：** 游戏完全使用 Java 编写，下载后是一个 JAR 包，可以通过 Java 命令行运行。

- • **核心玩法：** 玩家不仅需要建造炮塔来防御敌人，还必须规划完整的物流路线，确保传输带将资源运回核心，并为炮塔提供弹药和能量。

- • **扩展性：** 最突出的是它支持自建服务器和海量模组（Mods）。

### 6. AI 小说生成器
这款 AI 小说生成器被描述为一款“神级架子”，解决了 AI 写作中常见的剧情 Bug，例如主角眼睛颜色改变或遗忘反派名字等问题。

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrQAIibsNxLQiaJQWqibhMQ26UKiciaFRy82NFR8FpVr5xBkAwLnuvgiaYlre6Mp44YPP9tlrSDk0zWMbV5w/640?wx_fmt=png&from=appmsg)

- • **性质：** 完全绿色免费，双击即可运行。

- • **写作工具：** 提供了写小说所需的一切工具。

- • **创作流程：** 您可以先定义主题（例如“废柴小伙穿越白富美”），然后生成其整体架构，接着生成目录，最后进行定稿。

- • **质量保障：** 它提供“保姆级护航”，能够自动检测剧情中的 bug。

### 7. RSHB READER（RSS 订阅工具）
### ![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrQAIibsNxLQiaJQWqibhMQ26UKZyOsliaWDQvVElltcAm3UE5c6m9QKf3AJNjWOcv8gzfiaO0v7Epkbn7g/640?wx_fmt=png&from=appmsg)

这款工具旨在将全网的内容都变成您专属的信息流，让您足不出户即可纵览天下大事。

- • **内容聚合：** 不论是 B站的 UP 主更新、微博热搜、GitHub 的代码推送，甚至是小区停水通知，它都能将其转化为 RSS 订阅链接。

- • **使用方式：** 下载 RSHB READER 浏览器插件后，打开您喜欢的新闻或信息网站，插件会自动识别该网站中的 RSS 订阅点。复制找到的 RSS 链接并粘贴到任何您喜欢的阅读器中订阅，之后该网站的任何更新您都不会错过。

### 8. （coze Studio）
扣子是排名第一的智能体平台。Coze Studio 则将扣子平台搬进了您的本地环境中。

https://github.com/coze-dev/coze-studio

- • **开源内容：** 从模型管理到 GPT 引擎，从插件系统到部署工具，全部免费提供。

- • **部署：** 安装非常简单，通过复制运行命令、下载代码后运行即可。

- • **价值：** 这是一个企业级的智能平台，可以节省数十万的开发成本，让您在公司或实验室中快速搭建智能体网络。

### 9. 机器学习 Python 代码集合
这是最后一个项目，是一个精选的机器学 Python 代码集合。

- • **资源定位：** 为开发者和研究人员提供了一个全面、高质量且易于查找的工具资源。

- • **规模与分类：** 覆盖了 920 个开源项目，总计 510 万星标。共分为 34 个类别，包括机器学习框架、数据可视化、自然语言、图像数据以及时间序列分析等主流方向。

- • **评价：** 堪称学习 AI 的“葵花宝典”。
