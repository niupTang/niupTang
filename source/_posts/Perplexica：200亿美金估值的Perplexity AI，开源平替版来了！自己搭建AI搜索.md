## 🚀 项目简介：隐私安全的AI搜索解决方案
**Perplexica** 是一个基于人工智能的开源搜索引擎，被誉为"Perplexity AI的最佳开源替代品"。该项目在GitHub上已获得**超过14.5k星标**，最大的亮点是支持**100%本地化部署**，彻底解决数据隐私问题。

与需要将查询数据发送到云端的商业产品不同，Perplexica可以在你的本地服务器上完全运行，查询数据永远不会离开你的控制范围。这对于企业用户和注重隐私的个人用户来说，具有无可替代的价值。

## ✨ 核心功能：不输商业产品的完整体验
### 1. 🔍 多模式智能搜索
`# 支持的全场景搜索模式
search_modes = [
    "all",           # 全网搜索
    "academic",      # 学术搜索
    "youtube",       # 视频搜索
    "wolfram",       # 数学计算
    "reddit",        # 社区讨论
    "writing",       # 写作辅助
]`### 2. 🤖 多模型支持架构
- • **本地模型**：通过Ollama支持Llama 3、Mixtral等本地模型

- • **商业API**：兼容OpenAI、Anthropic、Groq等商业接口

- • **混合部署**：可根据需求灵活配置模型组合

### 3. ⚡ 实时网络检索
基于SearXNG元搜索引擎，能够从多个源头获取实时信息，确保搜索结果的时效性和准确性。

### 4. 📚 引用溯源机制
每个回答都附带原始来源引用，方便用户验证信息真实性，这一点完全对标Perplexity AI的核心体验。
