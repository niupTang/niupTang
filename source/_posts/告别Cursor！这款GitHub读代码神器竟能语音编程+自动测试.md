## 安装 Aider
安装 Aider 非常简单，只需运行以下命令即可：

`python -m pip install aider-install
aider-install`安装完成后，你就可以开始使用 Aider 了。

## 使用 Aider
### 进入你的代码库
首先，切换到你的项目目录：

`cd /to/your/project`### 配置 Aider
Aider 支持多种语言模型，你可以根据自己的需求选择合适的模型。以下是一些常见的配置方法：

使用 DeepSeek`aider --model deepseek --api-key deepseek=<key>`![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrQ3dibNgv0nuWJbZPEUMBSlKDCHLQiaICZVGiaias2pUkXsVH79wUsZyKNpnqtPEZ80rwyaGKURgERc5A/640?wx_fmt=png&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1&wx_co=1)

## Aider 的强大功能
### 支持多种编程语言
Aider 支持 100 多种编程语言，包括 Python、JavaScript、Rust、Ruby、Go、C++、PHP、HTML、CSS 等。无论是大型项目还是小型脚本，Aider 都能轻松应对。

### 自动提交更改
Aider 与 Git 集成，能够自动生成合理的提交消息并自动提交更改。你可以轻松区分、管理和撤消 AI 的更改，确保代码的版本控制始终处于最佳状态。

### 在 IDE 中使用
Aider 可以在你最喜欢的 IDE 或编辑器中运行。只需在代码中添加注释，Aider 就会根据你的需求进行更改，无需离开你的开发环境。

### 图片和网页支持
在开发过程中，你可能需要参考图片或网页。Aider 支持在聊天中添加图片和网页，你可以直接将屏幕截图、参考文档等添加到聊天中，为你的开发提供更直观的参考。

### 语音转代码
Aider 还支持语音转代码功能。你可以通过语音与助手交流，请求新功能、测试用例或错误修复，Aider 会根据你的语音指令自动实施更改。

### 自动检查和测试
每次 Aider 进行更改时，都会自动运行代码检查和测试。如果发现问题，Aider 会尝试修复，确保代码的质量始终处于最佳状态。
