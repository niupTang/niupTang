## 三、三步开启你的明星人生
### 小白友好型操作指南（Windows版）
1️⃣ **环境搭建**

`git clone https://github.com/hacksider/Deep-Live-Cam.git
cd Deep-Live-Cam
pip install -r requirements.txt`贴心提示：NVIDIA用户安装`onnxruntime-gpu`可提速300%

2️⃣ **模型配置**

- • 下载`GFPGANv1.4`与`inswapper_128_fp16.onnx`模型

- • 放置于`/models`目录下

3️⃣ **一键换装**

`python run.py --source 你的照片.jpg --execution-provider cuda`选择直播源→调整唇形同步→开启OBS推流，即刻变身！
