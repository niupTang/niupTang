* 戳上方蓝字“**牛皮糖不吹牛**”关注我

    大家好，我是牛皮糖！

  你是否曾因网页上层出不穷的广告而感到烦恼？是否在浏览网页时被各种弹窗打扰？今天，我要向大家介绍一个 GitHub 上拥有 26,642 星的开源项目——**hosts**，它能让你的网络环境瞬间变得清净。

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrR9Pwe4tH9FZatQwU2qX0o1sXyKbMj1sD6hUrHU0ahibwaZkQ3oOQQt1XUyQ8RcwCjC6xHX0ibibYRlQ/640?wx_fmt=png&from=appmsg)

**项目介绍：** 这个项目的核心是一个精心维护的**hosts**文件，通过修改本地 DNS 解析，将广告服务器的域名指向本地，从而在源头上屏蔽广告。这意味着，当你访问一个含有广告的网站时，这些广告将无法加载，给你一个无干扰的浏览体验。

除了基本变体之外，该存储库还提供 31 种不同的主机文件变体，包括或不包含统一主机。

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrR9Pwe4tH9FZatQwU2qX0o1QzIYeU7DHfXfoBgicWrl3jRJLkOSibLvNhjOk10sia382rUrK3CpZJkbg/640?wx_fmt=png&from=appmsg)

**如何使用：** 

- 1. 访问项目的 GitHub 页面。

- 2. 下载最新的**hosts**文件。

- 3. 将其替换到你的操作系统中的相应位置（例如 Windows 系统中的`C:\Windows\System32\drivers\etc\hosts`）。

- 4. 重启你的网络设备或计算机，使更改生效。

**项目特点：**

- • **全面性**：这个**hosts**文件集合了多个来源的数据，确保了广告屏蔽的全面性。

- • **更新及时**：项目维护者会定期更新文件，以应对不断变化的广告策略。

- • **易于使用**：即使是非技术用户，也能轻松地按照指导进行操作。
