* 戳上方蓝字“**牛皮糖不吹牛**”关注我

    大家好，我是牛皮糖！今天看到一款命令工具，AIChat 是一款功能强大的一体化 AI 命令行工具，集成了多种前沿技术与实用功能，适用于多种场景和用户需求。

![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrSMwE8h2m214wmRDfM7fMyID9HIJnGhNJPb76Yb9toLfPWKQgKicA0q6gx4k435Dexqn9ia0rOY7Aug/640?wx_fmt=png&from=appmsg)

### 核心功能
- • **多供应商集成**：AIChat 支持与 OpenAI、Claude、Gemini 等 20 多个领先的 LLM 供应商无缝集成，用户可以通过统一接口调用多种模型。

- • **命令行模式（CMD Mode）**：提供强大的命令行功能，用户可以通过命令行与 AI 进行交互，支持多种命令和参数配置。

- • **交互式 REPL 模式（REPL Mode）**：提供一个交互式的 Chat-REPL 环境，支持自动补全、多行输入、历史记录搜索、可配置的键绑定和自定义 REPL 提示符等功能。

- • **Shell 助手（Shell Assistant）**：通过自然语言描述任务，AIChat 可以将其转换为精确的 shell 命令，提高命令行效率，并能根据用户的操作系统和 shell 环境进行智能调整。

- 

### 高级功能
- • **多形式输入（Multi-Form Input）**：支持多种输入形式，包括标准输入、本地文件和目录、远程 URL 等，提供灵活的数据处理方式。

- • **角色定制（Role）**：用户可以通过自定义角色来调整 LLM 的行为，每个角色由一个提示和模型配置组成，以提高交互效率和生产力。

- • **会话管理（Session）**：通过会话保持上下文感知的对话，确保交互的连贯性。

- • **宏定义（Macro）**：将一系列 REPL 命令组合成一个自定义宏，简化重复性任务。

- • **检索增强生成（RAG）**：将外部文档集成到 LLM 对话中，以获得更准确和上下文相关的响应。

- • **函数调用（Function Calling）**：通过连接外部工具和数据源增强 LLM 的功能，扩展其核心能力，以处理更广泛的事务。

### 集成功能
- • **AI 工具（AI Tools）**：集成外部工具以自动化任务、检索信息和执行操作，直接在工作流中使用。

- • **AI 代理（AI Agents）**：作为 OpenAI GPTs 的 CLI 版本，结合指令、工具和文档来实现复杂任务的自动化。

- ![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrSMwE8h2m214wmRDfM7fMyICicxZk1K2EYX9CQVkibSlU8HzSLF5Z8ZXZA7VM5tHjXNxbFd6At4Cqgw/640?wx_fmt=png&from=appmsg)

### 本地服务器能力
- • **内置 HTTP 服务器**：AIChat 包含一个轻量级的内置 HTTP 服务器，便于部署，提供诸如 Chat Completions API、Embeddings API、Rerank API 等服务。

- • **LLM 竞技场（LLM Arena）**：一个基于 Web 的平台，用户可以在其中并排比较不同的 LLM。

- • **LLM 游乐场（LLM Playground）**：一个 Web 应用程序，用户可以直接从浏览器与支持的 LLM 进行交互。

- ![](https://mmbiz.qpic.cn/mmbiz_png/p1ESIQQvfrSMwE8h2m214wmRDfM7fMyI6j4SqWIDsE2xIxANUJle1BiabEQPXIutrDiaNlpSL2e1k1C8xftXJEow/640?wx_fmt=png&from=appmsg)

### 其他功能
- • **智能问答与推荐**：AIChat 能够通过自然语言处理技术，理解用户的问题并提供准确的答案，同时根据用户的历史对话记录进行个性化推荐。

- • **多语言支持**：支持多种语言的对话，帮助用户跨越语言障碍，实现更广泛的交流。

- • **文档与内容创作**：AIChat 可以帮助用户撰写论文、电子邮件、故事、诗歌等，还能进行语法纠错、信息检索和创意生成。

- • **智能办公助手**：在办公场景中，AIChat 可用于资料整理、会议纪要、办公自动化、企业知识库管理等，提升工作效率。

AIChat 的这些功能使其成为一个强大且灵活的工具。
